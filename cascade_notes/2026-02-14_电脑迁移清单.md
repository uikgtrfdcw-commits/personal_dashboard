# 电脑迁移清单
**日期**：2026-02-14
**用途**：换系统盘、换电脑、或任何需要重建环境时的检查清单

---

## 第一步：备份整个 CascadeProjects 文件夹

路径：`C:\Users\sweet\Downloads\CascadeProjects\`

拷贝到移动硬盘或U盘即可。文件夹内包含：

| 子文件夹 | 内容 |
|--------|------|
| `cascade_notes` | 结构化笔记、索引、用户偏好（**核心，丢了等于失忆**） |
| `articles` | 文章各版本（废墟上的算术题、伊朗遇刺、AI记忆、雄安等） |
| `Windsurf对话记录` | 原始对话存档（防止细节丢失） |
| `zhihu-data` | 知乎爬虫脚本、抓取的CSV数据、debug工具 |
| `personal_dashboard` | Streamlit 个人面板（合并版，含任务清单+健身计划） |
| `google_sheets_mcp` | Google Sheets MCP 服务端脚本 |
| `.windsurf/workflows` | `/restore` 等自动化工作流定义 |
| `.streamlit` | Streamlit 根目录配置 |

## 第二步：单独备份不在 CascadeProjects 里的文件

| 文件 | 路径 | 说明 |
|------|------|------|
| `secrets.toml` | `personal_dashboard/.streamlit/secrets.toml` | Google Sheets API 密钥，被 .gitignore 排除，**不会被 Git 推送**，必须手动拷贝 |
| `mcp_config.json` | `C:\Users\sweet\.codeium\windsurf\mcp_config.json` | Windsurf MCP 服务配置（GitHub Token、Google Sheets MCP 路径等），**在用户目录下，不在项目里** |

## 第三步：新系统上重新安装

- **Python**：安装后在项目根目录运行 `pip install -r requirements.txt`
- **Node.js**：MCP 服务需要，装完后在 `google_sheets_mcp/` 下运行 `npm install`
- **Git**：Streamlit 项目推送到 GitHub 用
- **Chrome + ChromeDriver**：知乎爬虫脚本需要，版本要匹配
- **代理**：`HTTPS_PROXY=http://127.0.0.1:17890`（端口视代理软件而定）
- **Windsurf**：重新安装，登录同一账号
- **知乎 Cookie**：旧的可能过期，重新登录浏览器获取

## 第四步：新系统上恢复工作

1. 把 `CascadeProjects` 文件夹放到 `C:\Users\sweet\Downloads\` 下
2. 把 `mcp_config.json` 放回 `C:\Users\sweet\.codeium\windsurf\mcp_config.json`
3. 把 `secrets.toml` 放回 `personal_dashboard/.streamlit/secrets.toml`
4. 安装 Windsurf，登录账号，打开 `CascadeProjects` 文件夹
5. **新开对话，输入 `/restore`**——这一步会自动读取索引和偏好文档，恢复全部工作上下文

## 不需要搬

- Windsurf 软件本身（重新安装即可）
- Windsurf Memories（跟账号走）
- 旧的对话记录缓存（不可靠，不要依赖）
