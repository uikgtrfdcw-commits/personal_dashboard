# 电脑迁移清单
**日期**：2026-02-14
**用途**：从主机迁移到笔记本（或任何新电脑）时的检查清单

---

## 必须搬的文件夹

| 文件夹 | 内容 |
|--------|------|
| `cascade_notes` | 结构化笔记、索引、用户偏好 |
| `Windsurf对话记录` | **新增**：原始对话存档（防止细节丢失，如2.28运动计划） |
| `articles` | 文章各版本（厨房比喻、AI记忆、雄安等） |
| `zhihu-data` | 知乎爬虫脚本、抓取的CSV数据、debug工具 |
| `personal_dashboard` | Streamlit 个人面板（合并版，含任务清单+健身计划） |

以上均在 `c:\Users\sweet\Downloads\CascadeProjects\` 下，打包拷走即可。

## 新电脑上需要重新配置

- **Python**：安装后 `pip install -r requirements.txt`（每个项目各有一份）
- **Chrome + ChromeDriver**：知乎爬虫脚本需要，版本要匹配
- **Git**：Streamlit 项目推送到 GitHub 用
- **代理**：`HTTPS_PROXY=http://127.0.0.1:17890`（端口视新电脑代理而定）
- **Windsurf**：重新安装，登录同一账号
- **知乎 Cookie**：旧的可能过期，到时重新登录浏览器获取

## 不需要搬

- Windsurf 软件本身（重新安装）
- Windsurf Memories（跟账号走，但内容有限）
- 对话记录（可能有本地缓存，但不要依赖）

## 到新电脑后的第一步

1. 把五个文件夹放到 Downloads 目录下
2. 安装 Windsurf，登录账号
3. 新开对话，让 AI 先读 `cascade_notes/_index.md`
4. 再让 AI 读 `cascade_notes/2026-02-14_用户偏好与工作方式.md`
5. 开始干活
