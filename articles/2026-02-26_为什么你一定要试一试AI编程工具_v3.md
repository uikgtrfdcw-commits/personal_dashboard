# 为什么你一定要试一试 AI 编程工具（v3）
**日期**：2026-02-27
**底本**：用户修改版板块1 + 板块3中不重复的内容（执行力对比、记忆力对比）融合；去掉拖延症、长期记忆、固化流程、工具分类等板块；融合部分加粗标注

---

我们习惯把技术分成两类：一类是给专业人士用的，一类是给普通人用的。编程显然属于前者。它属于那个我们心照不宣地绕开的世界。就像高等数学、量子物理、或者修下水道，我们默认那是专业人士干的事情。

但偶尔会有一些时刻，某种技术从前一类悄悄滑入后一类。打字机曾经是专业排版工人的工具，个人电脑曾经是极客的玩具，Excel 刚出来的时候人们觉得只有会计才用得上。每一次滑动发生的时候，大部分人的反应都是一样的：跟我没关系。然后过几年回头看才发现并非如此。

我怀疑 AI 编程工具正处在这个滑动的临界点上。

之所以说是怀疑，而不是确信，是因为我自己也是最近才被说服的。我对编程的全部认知，长期停留在大学时代 C 语言作业的创伤性记忆里——枯燥、无趣、没用。一个分号的位置不对，整个程序就罢工了，而你完全不知道为什么。那种体验让我形成了一个根深蒂固的信念：代码是一种需要特殊才能的语言，就像绝对音感或者空间想象力，有的人天生就有，没有的人再努力也够不着。

直到上个月，我开始用 Windsurf。

我当时正在被琐碎的事务性工作困扰。那种你明知道毫无创造性、纯粹是机械重复、但又不得不做的事。我至今相信，如果人类文明最终毁灭，原因不会是核战争或气候变化，而是无数人把无数个小时花在了这类劳动上，然后集体丧失了对生活的全部热情。

然后我试着向 Windsurf 描述我的需求。

然后我看着它开始写代码。那种感觉很奇怪——就像你对一个外国工匠比划了两下你想要的柜子形状，然后他转身走进车间，锯子响了十分钟，出来的时候手里真的端着一个八九不离十的柜子。他说的是啥你一个字也没听懂，但他做出来了。

我当时的感受，说实话，不是兴奋，是困惑。这就行了？就这么简单？那我之前花掉的那些时间算什么？

我又用 Windsurf 做了很多事：网上抓取数据、数据处理自动化。如果你跟半年前的我描述这些，我都会觉得你在开玩笑。我连 Python 和 Java 的区别都说不清，我怎么可能在做这些事？

这暴露了一个人们很少怀疑的假设：我们总以为"能力"是一种个人属性，是你经过长期训练后内化的东西。你会弹钢琴，你会写代码，你会做手术。但 AI 编程工具提出了一种不同的可能性——能力不一定要内化，它可以是外部化的。你不需要"会"编程，你只需要能准确地描述你的问题。描述问题的能力，和解决问题的能力，被拆开了。

这个拆分的意义比看上去要大。在人类历史的绝大部分时间里，这两者是绑在一起的。你得先成为铁匠，才能打出一把好刀。你得先学会裁剪，才能做出一件衣服。"知道自己要什么"和"能把它做出来"之间，隔着至少数年的专业训练。AI 正在把这个间隔压缩到几分钟。

**你可能会说：我已经在用 ChatGPT 了，AI 编程工具有什么不一样？**

**区别在于：聊天工具只能告诉你怎么做，编程工具能替你做。**

**比如我之前用 Gemini 管理工作计划。它确实很强——拆解任务、排优先级、做复盘，条理清晰。但到了执行环节，所有的脏活累活还是要我自己干。我说"帮我把任务 1.1 标记为已完成，移到归档表"，它会认认真真告诉我该怎么操作。然后我得自己打开表格，找到那行，手动改状态，手动移过去，手动调格式。它是一个很好的教练，但不能亲自动手。**

**而 Windsurf 直接就帮我改好了。不是给我一段教程，而是它自己连上表格，找到那行数据，移过去，排好序，格式化好，然后告诉我"搞定了"。你可以一口气提很多要求，说完一段话的功夫，它把你可能折腾半小时的操作全做完了。整个过程更像是在指挥别人工作——你只需要下命令、给结果提意见就行了。**

**它甚至比你自己更记得你做过什么。**

**Gemini 的记忆本质上是一种"印象"：时间一长，细节就丢了。就像凭回忆复述上周的会议内容，大意能讲对，但具体谁说了什么，经常搞混。而 Windsurf 的每一次操作，都会变成本地文件里的真实记录——代码改了哪一行、文档加了哪句话、什么时间做的，全都白纸黑字存在你的电脑上。我和它复盘上周的工作时，经常感到惊讶：很多细节我自己都忘了，它却记得清清楚楚。**

当然，这种压缩是有边界的。我让 AI 造的东西，都是"够用就行"的小工具。如果你需要建一个服务几百万用户的系统，AI 仍然无能为力。可靠性、安全性、可维护性，这些仍然需要专业开发者。但问题是，大多数人大多数时候需要的并不是一栋大楼，而是一把顺手的螺丝刀。

我现在回想过去自己上半辈子的工作，感觉心情很复杂。一方面觉得浪费了很多时间，另一方面又意识到，在你不知道一件事可以被自动化之前，你甚至不会觉得手动做有什么问题。你以为那就是工作的样子。就像在洗衣机发明之前，没有人觉得手洗衣服是一种"低效"，那就是洗衣服。
