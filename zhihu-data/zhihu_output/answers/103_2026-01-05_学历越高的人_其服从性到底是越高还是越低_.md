# 学历越高的人，其服从性到底是越高还是越低?

- 日期: 2026-01-05
- 赞同: 0
- 评论: 0
- 链接: https://www.zhihu.com/question/1989275249007678149/answer/1991653000083686810

---

这是一个非常有意思的各种矛盾交织的话题。作为一个在互联网行业摸爬滚打多年的算法工程师，我也经常在招聘、带团队以及和各路大厂的一颗颗螺丝钉打交道的过程中思考这个问题。
大家对于学历和服从性关系的认知偏差，其实源于我们混淆了两个完全不同的概念。我们在谈论 
学历
 的时候，绝大多数时候谈论的是 
筛选机制下的做题能力
，而只有极少数时候，是在谈论 
真正的学术素养和批判性思维
。
这就像我们在训练一个深度学习模型。你是希望这个模型在训练集上表现完美，也就是完全拟合标准答案，还是希望它具有强大的泛化能力，能够处理从未见过的复杂情况？目前的教育体系，绝大多数时候是在批量生产前者，也就是大家常说的过拟合模型。
我们先把结论放在这里：
在现行的教育体制下，本科学历往往代表着服从性的巅峰；而当你突破了某个临界点，真正进入知识的无人区，比如读到顶尖博士或者成为真正的学者，服从性会断崖式下跌。
为什么这么说？我试着用我写代码和看人的经验，把这个逻辑层层剥开来讲。
我们从小到大经历的教育，本质上是什么？
我相信很多人读过米歇尔·福柯的 
规训与惩罚
。虽然这本书很难读，但里面的核心逻辑其实非常贴近我们的现实生活。福柯认为，现代学校、军营、监狱、工厂，在建筑结构和管理逻辑上具有惊人的相似性。它们的目的只有一个，就是通过空间隔离、时间规划和行为规范，把人训练成 
温顺的肉体
。
从小学到高中，甚至到大部分本科教育，我们在做什么？我们在寻找标准答案。
老师告诉你，这个公式是对的，那个历史事件的意义是这样的，阅读理解的主旨是那样的。你不需要去质疑为什么，你只需要记住，并且在考试的时候把它精准地复现出来。
能够考上985、211的学生，或者是现在的双一流大学生，本质上是什么人？是这套规则下最完美的执行者。他们最擅长的事情，就是揣摩出题人的意图，压抑自己的个性，长时间集中注意力去完成一个并非自己内心渴望的任务。
这在算法里叫什么？这叫 
强监督学习
。输入和输出都有明确的标签，只要你的输出和标签不一致，Loss Function 就会给你巨大的惩罚（扣分、排名下降、老师家长的责备）。经过十几年的这种训练，一个人最本能的反应就是：
标准在哪里？权威在哪里？我该怎么做才能让上面满意？
所以，知乎上说大学生最听话，工作以后老老实实当牛马，这完全符合逻辑。
这帮孩子太好管了。你给他们一个KPI，不需要告诉他们这个KPI背后的业务价值是不是扯淡，他们自己就会卷起来。他们会为了那一点点绩效的提升，为了年终评级的一个A，主动加班，主动优化流程，甚至主动自我PUA。因为这就是他们过去十六年最擅长的事情——在既定的规则内把分数刷到最高。
我在大厂带过很多校招进来的应届生。这些孩子非常优秀，代码写得漂亮，文档写得规范。但是你一旦问他：
这个需求真的是用户需要的吗？这个算法上线后的伦理风险你考虑过吗？
 他们往往会愣住。因为在他们的潜意识里，需求是 
题目
，是不可质疑的输入，他们只负责解题。
这种高学历，其实是 
高服从性
 的另一种说法。这也是为什么大厂喜欢招高学历的年轻人，不仅仅是因为他们聪明，更因为他们 
好用
。他们是工业化大生产中最精密、最耐用、也最听话的零件。
既然大学生这么听话，为什么部队里的老班长反而觉得大学生刺头，不如高中生好管？
这里就涉及到一个非常有意思的心理状态，我称之为 
小布尔乔亚式的矫情
，或者叫 
半吊子的启蒙
。
大学生虽然在行为模式上被规训了，但他们毕竟读了几本书，接触了一些自由主义的思想，看过一些国外的电影。他们虽然没有形成独立的思想体系，但他们形成了 
自尊心
 和 
自我意识的萌芽
。
这种自我意识非常尴尬。它不足以支撑他们去进行深刻的逻辑反抗，但足以让他们对机械的、粗暴的命令产生本能的抵触。
高中生或者更低学历的人，进入部队，他们往往是一张白纸。在他们的认知里，力量就是权威，班长比我强，我就听班长的。这是一种朴素的生存逻辑，非常直接，管理成本很低。
但是大学生不一样。大学生会想：
凭什么？
你让他叠被子，叠成豆腐块。高中生会觉得这是任务，练就是了。大学生会想：
这有什么实战意义？这是形式主义。我是一个有独立人格的知识分子，为什么要在这里做这种毫无意义的体力劳动？
注意，他们虽然心里这么想，但他们通常不敢正面硬刚（因为多年的做题训练让他们害怕惩罚）。于是他们就会表现出一种 
消极抵抗
。动作慢吞吞，眼神里带着不屑，私底下讲怪话，对于命令执行打折扣。
这就是管理者最头疼的状态。你没法因为他彻底不执行而惩罚他，因为他也在做，但他做得让你极其难受。他的内心并没有真正服从你，他只是在敷衍你。
这种 
眼高手低
 的状态，就是部队觉得大学生不如高中生好用的原因。他们失去了一张白纸的纯粹，却又没有获得真正的智慧。他们卡在了中间，既不够愚钝，又不够通透。
这就好比我写程序。最怕的不是完全跑不通的代码（那是Bug，修就是了），也不是运行完美的代码。最怕的是那种 
能跑，但是偶尔会报几个莫名其妙的Warning，占用资源奇高，还很难重构
 的代码。这就是普通的大学生在强力执行机构眼里的样子。
那么，学历再往上走呢？
当我们谈论真正的顶尖学者，那些读到博士，或者在某个领域钻研极深的人，情况就完全反转了。
真正的学术研究，要求的是什么？是 
创新
。
创新的前提是什么？是 
质疑
。
你要发表一篇顶级的Paper，你必须证明前人的方法有缺陷，或者目前的SOTA（State Of The Art）模型还有提升空间。你不能顺着别人的话说，你必须挑刺。你必须具备极强的 
批判性思维
。
这种训练，会彻底重塑一个人的大脑。
在我的算法团队里，最难管理的往往是那些顶尖名校出来的博士，或者是资深的架构师。你给他们下达一个行政命令，比如 
今年要把某个指标提升20%
。
刚毕业的本科生会马上打开电脑开始调参。
而那些资深的博士会直接把电脑合上，问你：
这个指标的定义是不是有问题？根据数据的分布，提升20%违背了统计学规律。如果是为了向上管理，我不做。如果是为了业务价值，我们需要重新定义问题。
你看，这就是 
最高级的低服从性
。
他们不是在撒泼打滚，不是在偷懒耍滑。他们是在用 
逻辑
 和 
知识
 瓦解权威的合法性。
这种反抗是最致命的。因为领导没法用威权来压服他们。在绝对的逻辑和数据面前，职级是没有意义的。
汉娜·阿伦特在 
平庸之恶
 里分析过艾希曼。艾希曼不是恶魔，他只是一个 
不再思考
 的官僚。他放弃了判断力，把自己变成了一个执行命令的机器。阿伦特认为，这种 
无思
 才是最大的恶。
而真正的知识分子，真正的最高学历拥有者，他们最大的特征就是 
无法停止思考
。他们时刻在审视：
这个命令合理吗？这个流程正义吗？这个系统的终极目标是什么？
当一个人具备了这种审视能力，他就永远无法成为一个顺从的 
工具人
。他可能在表面上会保持礼貌，但在核心问题上，他寸步不让。
这也是为什么很多大公司到了最后，技术大牛和管理层往往会有冲突。因为管理层追求的是 
效率和控制
，是把人变成资源；而技术大牛追求的是 
真理和逻辑
，是对客观规律的尊重。
所以，回到最初的问题。
学历高的人，服从性到底是高还是低？
这取决于你定义的 
学历高
 到底处于哪个阶段。
如果是处于 
应试教育的顶点（本科/硕士阶段的做题家）
，他们的服从性是最高的。因为他们就是被筛选出来的最听话、执行力最强的一群人。他们是现代社会的 
完美燃料
。
如果是处于 
探索知识的顶点（真正的学者/思想者）
，他们的服从性是最低的。因为他们已经看透了规则的本质，他们只服从于真理，而不服从于人。
我们现在的社会环境，往往混淆了 
受过教育
 和 
有智慧
 这两个概念。
拿到毕业证，只是证明你受过 
职业规训
。这不代表你有独立思考的能力，甚至在某种程度上，这代表你的独立思考能力已经被系统性地阉割了，换取了高效的执行能力。
而真正的智慧，往往需要经历一个 
再社会化
 或者 
反向重构
 的过程。
你需要把你脑子里那些 
标准答案
 统统倒掉。你需要意识到，这个世界是草台班子搭建的，很多权威也是在那装模作样。你需要明白，所谓的规则，很多时候只是既得利益者为了降低管理成本而设立的防火墙。
我在行业里呆了这么久，见过无数聪明人。
我见过清北毕业的卷王，在毫无意义的PPT汇报里耗尽心血，只为了老板的一个点赞。这种人，学历极高，但本质上是 
高级奴隶
。
我也见过二本毕业但野蛮生长的极客，敢于在全员大会上指出CEO的逻辑漏洞，并且给出了可行的替代方案。这种人，学历看似不高，但他是真正的 
自由人
。
这篇回答不是为了贬低大学生，也不是为了神话博士。我想说的是，我们每个人都需要警惕那种 
无意识的服从
。
作为算法工程师，我每天都在调教模型。我知道，一个过拟合的模型虽然在训练集上 Loss 很低，但在真实世界里往往一无是处。
人也一样。
如果你发现自己只是在不断地满足别人的期待，在不断地寻找标准答案，在害怕偏离轨道，那么无论你手里拿的是什么文凭，你都只是一个 
高服从性的零件
。
真正的反抗，不是去部队里当刺头，不是在网上当键盘侠骂街。
真正的反抗，是 
保持思考
。
是在接到每一个指令的时候，在脑子里多跑一个线程：
为什么？合理吗？有更好的解法吗？
是在面对权威的时候，不卑不亢，用逻辑而不是情绪去对话。
是在所有人都在盲目狂奔的时候，敢于停下来，看看方向对不对。
这种能力，学校不会教给你，甚至会刻意压制你。但这才是区别人才与耗材、智慧与学历的根本分界线。
希望看这篇回答的朋友，无论你现在是什么学历，都能跳出 
做题
 的惯性。世界不是一张试卷，没有标准答案。只有当你开始质疑题目本身的时候，你才真正开始了自己的生活。
不管你是做算法的，还是送外卖的，保持这点 
逻辑上的不服从
，是你作为一个人，最高贵的底色。
