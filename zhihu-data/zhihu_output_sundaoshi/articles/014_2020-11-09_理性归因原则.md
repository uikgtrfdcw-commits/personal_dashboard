# 理性归因原则

- 日期: 2020-11-09
- 赞同: 0
- 评论: 0

---

读到了一句很有意思的话——人要分清三件事：
别人的事、自己的事、老天爷的事
。我个人称之为理性归因原则。我认为搞清楚这个原则，对于应对生活中的困难和挫折一定大有益处。
面对挫折的时候，人们通常有两种极端反应，一种是恐慌，自我怀疑，觉得天塌了。尤其像我这种自信心不足的弱鸡，一旦事情搞砸了，或者别人对我发脾气，就很容易陷入自我怀疑的陷阱。 
自我怀疑当然也不全是坏事。俗话说，如果你从来不觉得自己是个XX，那么很可能情况已经不容乐观。然而在矛盾爆发的紧要关头，过度的自我怀疑只会让你思维混乱，迟迟无法找到问题的最优解。 
另一种极端的反应是愤怒，怨天尤人，觉得别人是猪队友，觉得世界辜负了自己。这当然能够让自己感觉好过一点，但显然并不利于解决问题本身。 
如果你也有过这两种反应，那么理性归因原则就很有用了。说到底，挫折发生的可能原因无非三种：自己的问题，别人的问题，老天的问题。显而易见，你需要专注于自己的问题。 
但现实情况是，我们往往会反着玩，把自己的事不当回事，把别人的事和老天的事当成天大的事。这就是典型的非理性。 
而理性的分析是长这样的： 
别人很生气/发挥的不好？这是别人的事，和我无关；运气不好？这是老天爷的事，和我无关；我自己的发挥怎么样？下一步应该怎么做？
如果我发挥不佳，但结果很棒，那么我应当反思。如果我完美发挥，但结果很糟，那么我应当为自己感到骄傲。 
当然，我并不是说单凭自己的努力就能改变一切。我是说在试图解决问题的时候，不要浪费时间在不可控的因素上。一行法师说，事情如果能解决，那么根本不必担心。不能解决，那么你担心也没有用。 
理性归因就是抓住重点，搞清楚事情发生的根本原因在哪里，就是放弃沉没成本，面向未来，专注于如何解决问题。搞清楚这个原则，能够帮助我们排除情绪干扰，迅速理解当前的处境，定位下一步行动。同时，这也极大的减轻了认知负担——准确来说，三分之二的负担。
…
理性归因是一种积极的心态，意味着无条件的自我肯定。
在这种心态下，失败永远只是能力和方法的问题，而不是自身人格的问题。
换句话说，只有积极的自我认同，才能作为稳固的心理基石，并在此之上，建立理性乐观的行为逻辑。
一个心理内核稳定的人，遇到问题就事论事，归因而不归罪。他可以很自然的反思自己的行为，而不会彻底否定自己的人格。这样的人拥有足够的自信心和安全感，能够坦然的承受失败，抗压能力也比较强。
…
最近我发现，自己常会思考过去的错误，产生强烈的懊悔情绪，而不是当下自己能够改变的问题，这也是一个非常不健康的思维方式，过去的事情也是老天的事情。
最后，人们总是把失败归因于环境，把成功归因于自己。这里主要讨论的是前一种情况。对后者感兴趣的同学，请移步Michael Lewis的毕业演讲
『不要吞吃命运的饼干』
。
