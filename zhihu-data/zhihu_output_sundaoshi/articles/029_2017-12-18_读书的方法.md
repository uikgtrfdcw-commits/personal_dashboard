# 读书的方法

- 赞同: 0
- 评论: 0

---

台湾人管读书叫“念书”，有调侃的味道。我欣赏这种态度。又不是为中华之崛起，何必一本正经。
所以很少在地铁里捧起一大部头来读，那是聪明人的高雅事儿，而我向来不惮以最大的恶意揣测自己的智商，只好看看手机。
很惭愧，我很少大段时间读书，闲暇时候总是无所事事。大概是小时候管的严，缺少自由支配的时间，有空只能偷偷看看电视，注重一时的刺激，结果长大后有了余暇，也不喜欢花大段的时间来读书了。
正所谓『稀缺心态』是也。
还好百度拯救了我。由于搜索引擎的存在，获取信息的门栏被大大降低了。不管你对什么感兴趣，只要有时间有耐心，总能在互联网的犄角旮旯里找到你想要的东西。互联网太大，而且越来越大。人类从来未面对这般无边无际的信息海洋。
不过红楼梦说得好，大有大的难处。信息多了，信息获取成本却也在增加。比如很多人有这个体会：正在看一个网页，看到一半对另一个话题产生了兴趣，随后又点开了作者主页和他关注的链接... 最后对着密密麻麻的网页标签，都不晓得自己身在何处，该看哪里。
一个应对方法是做笔记。这样就知道自己看过了什么，没看过什么，该看什么，从哪看起。 钱钟书说做笔记的时间应该多出读书一倍，能用自己的话表达出来的，才是自己的东西。
再就是多读那些建立逻辑，观其大略的文章。 反之，如果一篇文章事无巨细，啰哩啰嗦，原来的问题还没解决，就被一大堆新问题淹没了。《南征北战》说的好，不要纠结一城一池的得失。
喜欢《晓说》的朋友往往有这个体会：单单一期节目，从大航海时代，工业革命，讲到科举制度，地缘政治，谈笑间就把原本支离破碎的知识点，连接成了完整的因果关系，让人听完心头雪亮，甚至顿悟。
这时才意识到，知识是千丝万缕，纷繁驳杂的蜘蛛网。只要理清脉络，抓住主干，擅长联想的人类大脑就会自动把知识编织起来。
还有一个常见的问题是读不进去。这其实很像小时候上课听不进去。讲的太慢，大脑会走神。讲的太快，大脑会消化不良。
所以作者的节奏和读者的节奏要尽量吻合。琴瑟相谐最性福嘛。初学者去读专业论文，院士去读入门教材，肯定都昏昏欲睡。
另外，我发现自己在网上写东西时，会比平时花多得多的功夫去查证，润色。可见虚荣心真的是好朋友。锦衣夜行，无人欣赏，不但孤陋而寡闻，更少了很多生活的乐趣。
公西华和孔子在一起谈论志向，说自己希望春天时能和一群陌生人去河里洗澡，然后唱着歌儿开心的样子。
这是我的志向：
周末的早晨，和几个陌生人窝在咖啡馆里念书。在静止而温馨的时间里，大家抬头对视，心照不宣。
