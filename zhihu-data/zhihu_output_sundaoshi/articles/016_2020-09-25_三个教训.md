# 三个教训

- 日期: 2020-09-25
- 赞同: 0
- 评论: 0
- 链接: http://zhuanlan.zhihu.com/p/259405861

---

我有个坏习惯。遇到问题一想不明白，一有压力，就会拿起手机，往脑子里面灌输信息，然后思路就全没了。
为此我深感头痛，只好常常躲到楼下咖啡馆，坐在视野通畅的角落里，遇到问题想不通的时候，第一反应是抬头看外面的景色，而不是打开手机。这样就不会打扰原有的思路。
于是我得到了一个教训：
摄入信息要小心。
不是主动想到的信息，那就要敬而远之。否则，内心的想法就会被淹没。
遇到压力，停下来看窗外是可以的，因为思路还在继续。一旦停下来刷视频，完了。你已经想不起来本来要干什么了。
顺便，我觉得这也是开放型办公室的最大问题。很多时候我们讨厌的不是工作，而是每隔五分钟就有人过来跟你交流工作。导致你根本无法全心投入任何事情。
当然，这些都已经是我的老毛病了。翻了翻三个月前的一篇日记，内容和今天是一毛一样的… 
说到底，这些都是因为习惯。我们总会忘记当下的教训，然后自动捡起那些对自己有害的，但从小养成的习惯。它们已经成了本能，一不小心便旧病复发。
人是容易忘记的动物。我们不但会忘记，而且会忘记自己曾经忘记。生命中的任何一段经历，即使当时再刻骨铭心，如果没有及时沉淀，也一定会像雪地上的脚印，慢慢消失掉。
于是我得到了第二条教训：
知道了不行，熟悉了也不够，要真正进入潜意识，成为条件反射才可以。
一个观念需要来回反复的反思实践，才能真正抵达意识深处。就像秦汉帝国，至少经历了三次的全面战争，才算彻底改掉分封制的毛病。
于是我决定把这些教训写下来，常常提醒自己。虽然可能重蹈覆辙，但至少每次都会发现更多的细节，感受似乎也更深一些。
就像现在，只要我一拿起手机，就立刻会联想到咖啡馆外的明媚阳光，以及夏日的午后，抹茶冰激凌在舌尖慢慢融化的味道，然后我就赶紧滚去咖啡馆了。
所以写东西真是个好习惯。那些重要的心得体会、以及让自己感动的瞬间，写到纸上，才能活的更久一些。
这也是我得到的第三个教训：
阳光、午后、抹茶冰激凌。
这些都是细节，魔鬼都在细节里。理性分析无法定位这些细节，更无法共情。如果你要真正理解一件事，或者帮别人理解，理性分析是不够的，还要把内心带入到当时的体验和状态，感同身受才行。会心一笑更好。
