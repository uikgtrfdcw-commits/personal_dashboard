# 从中英夹杂的现象谈起

- 日期: 2017-09-06
- 赞同: 0
- 评论: 0

---

人们常会反感中英夹杂的表达方式，认为那是哗众取宠，崇洋媚外，甚至对华夏文明的大不敬。
可多语言混杂的现象，在历史上是广泛存在的。比如诺曼征服时期，英国人各阶层普遍流行讲不同程度的法语。不久以后约定俗成，新的英文演化成一种英法交杂的语言。
语言的本质是思维的表达。你的思维在哪里，你的语言就在哪里。无意间的中英夹杂，意味着局部英文思维的形成。这就像学说话的婴儿在咿咿呀呀中，偶尔夹杂一句“妈妈”一样，是自然而然的，不必大惊小怪。
婴儿就不会大惊小怪，婴儿没有对错的观念（在父母足够耐心的情况下），不怕犯错，所以婴儿的语言学习能力是一流的。等长大进入学校，又进入社会，就会逐步丧失这种能力。事实上，社会的道德评判越是锋利严苛，学习能力的退化就越厉害，而语言本身也就越简陋贫乏。
…
我们都有自己独一份的语言特征，中英混杂不过是其中一种。不信你仔细观察，年轻人，老年人，不同行业，不同爱好的人，都会自发形成内部独立的小型语言体系。这些小圈子中经常会流传一些精妙有趣的表达，圈内人听了会心一笑，圈外人听了一头雾水。
语言只是观念的容器而已。
说到底，学习新的语言，就是学习一种新的思维方式。有人觉得背单词书是浪费时间，这是很有道理的。一个词汇单独拿出来没有任何意义，不构成任何思维。背单词不是学习语言，而是学习符号。这些松散机械的符号应付考试尚可，但对沟通交流几乎没有帮助。
所以程秧秧说，从爱好入手，是学习一门外语的不二法门。如果你经常看NBA原版比赛，那么在球场上和外国人简单交流就不成问题。如果你是二次元爱好者，随口绉几句日文更是自然而然。因为爱好顺便学好一门外语，这样的例子实在太多啦。
再者，一门语言也并非铁板一块，而是一个纷繁庞杂的思维系统。它的各个细分领域，各个方言，都有自己独特的表达习惯和思维方式。新闻联播中文和德云社中文差异巨大，政府报告英语和哈利波特英语也是两码事。作为一个非专业语言爱好者，能够做到精通一门语言的一个方面，一个话题，就已经是非常了不起，并且切实可行的成就啦。
当然，也并不是所有人都要去学什么外语。很多抽象思维发达，动手能力超强的孩子，在外语课上却一窍不通。如果不考虑应试因素，这些孩子花在英语课上的时间，还不如用来学微积分。
毕竟，数学才是物理世界的语言。
