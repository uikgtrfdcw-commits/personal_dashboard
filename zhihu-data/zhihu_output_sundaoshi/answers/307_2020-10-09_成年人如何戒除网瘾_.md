# 成年人如何戒除网瘾？

- 日期: 2020-10-09
- 赞同: 0
- 评论: 0
- 链接: https://www.zhihu.com/question/20365427/answer/1515023272

---

网瘾是为了缓解某种痛苦。
其实不只是网瘾，任何让人上瘾的东西，无论是毒品、赌博、酒精，还是是性、剁手、甜食，甚至工作，都可以被用来缓解痛苦。很多毒品本来就是止痛剂。
这种痛苦可以是肉体上的，也可以是精神上的。而大脑对于二者的感知非常类似。一个人失恋的感觉就像被刀子捅了一样。他会使用一切手段，不顾一切后果，试图逃离这种痛苦。
『我所有的这些扭曲行为，都只是为了忘记几个小时而已』 ——Keith Richard，滚石吉他手，重度海洛因成瘾者。
成瘾的原因并不是某些专家所谓的基因遗传、意志力薄弱，成瘾就是因为痛苦。真正问题并不是
『如何戒除网瘾』，
真正问题是
『如何化解痛苦』。
①
拿我来说吧。 
我也是在大学『染上网瘾』的，程度比题主有过之无不及。大一的时候，我突然发现自己看不懂教材。人家随便看看书就能90分，我拼了老命也就是个及格。
现在想想也没啥大不了的，术业有专攻，无非就是对理工科不开窍呗。 但我当时对自己期许甚高，深信“没有努力不能解决的问题”。问老师，问同学，把学霸同学的作业偷偷拿出来研究。努力了大半年，毫无长进。 我性格比较差，同学关系处理的不是很好。加上一些其他挫折，很快就崩溃了。
我开始玩游戏，给自己『减压』。开始是去学校机房，后来是去网吧，再后来是通宵去网吧。我生活迅速被游戏淹没。
我感到恐惧。 像所有决定痛改前非的瘾君子一样，我下定决心，再也不玩了。但你又如何面对现实的噩梦？太痛苦了。看不到出路。这个时候你转念一想：只要打开显示器，一切烦恼便烟消云散。 
你不是喜欢盯着显示器，你只是不喜欢看到这个糟糕的自己。
曾经尝试过一切可能的戒瘾方法：做计划、去自习室、聊天、跑步、绝食、删游戏、卖电脑、断网、赌咒发誓、悬梁刺股，我想当时就算有电击器我也会买来试试的。然而并没有鸟用，每次都是坚持一段时间，然后成功复吸。 这种事情没法跟家里说，因为只会被痛骂一顿。也没法找老师帮忙，谁有时间给你从头讲起？
差生难做。过来人都懂。
其实当时班级里有很多爱玩游戏的同学。但人家是游戏学习恋爱保研出国样样不耽误，最可气的是游戏玩的还比你好，真是岂有此理。当然也有和我一样悲剧的。大二系里面跟我一起降级的有七八个，据我所知最后都没有拿到毕业证。可见高考春风得意，本科沉迷游戏这种剧情，在大学里其实非常普遍。 
看出来了吗？我当时真正的痛苦是缺少『
陪伴
』和『
自尊心的满足
』。我当时亟待解决的问题不是如何不玩游戏
，
而是『
如何追到妹子
』，以及『
如何弄懂这本该死的电磁场教材
』。如果你搞错了方向，怎么努力都是浪费时间。
当然了，感情和学业是很多人都面临的共性问题，很难在一个回答里说清楚。再说生活不易，每个人面对的问题也未必相同。
不过这么多年以后，如果让我重新上一遍大学，我的玩法会完全不同。这里写下来，希望对题主有所启发：
1. 
转专业。
糟糕的表现往往是因为被放在了糟糕的位置。
2. 
不要过分责备自己。
世俗意义上的成功是诸多变量综合作用的结果，至少你爸妈的责任比你大多了。读一读马尔科姆的《异类》。
3. 
顺其自然，接纳自己的不完美。
推荐一下武志红早期的文章。②
3. 
自己做决定。
就像捡到的钱不会珍惜，别人安排的时间，你是不会认真对待的。
4. 
试试咖啡馆和自习室。
这两个地方有疗愈BUFF，专治重度死宅。 
5. 
在兴趣爱好上花时间。
热爱自己专业的毕竟是少数。当你对自己的本职工作感到失望，副业就是你最大的寄托。
