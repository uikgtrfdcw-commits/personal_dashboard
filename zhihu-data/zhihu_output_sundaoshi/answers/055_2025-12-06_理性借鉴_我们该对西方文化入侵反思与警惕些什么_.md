# 理性借鉴，我们该对西方文化入侵反思与警惕些什么？

- 日期: 2025-12-06
- 赞同: 0
- 评论: 0

西方总是向中国输出最糟糕的思想。比如激进的平等主义、人口控制论、以及最新的例子——现代货币理论（MMT）。
MMT的核心观点是：拥有主权货币的政府不会破产，可以通过印钞来解决就业和债务问题。操作上，MMT主张央行直接购买国债。简单来说，就是超发货币。
在当下，中国正面临通缩压力、地方债危机和房地产泡沫，国内也出现了"财政赤字货币化"来救市的声音，让政府以为可以不受限制地花钱。然而，就像之前的计划经济和人口控制一样，MMT提供了一种看似完美的技术性解决方案，最终却可能导致恶性通胀和货币信用的崩塌。
有趣的是，无论美国还是中国，在名义上都坚决否认在搞MMT，但在实质操作上，两国都在摸着MMT过河。
如果你问美联储主席鲍威尔或财政部长贝森特，他们会坚决告诉你："我们不信奉MMT，财政赤字很重要。"但在实际操作中，美国已经在事实上执行了MMT的某种变体。美国现在的债务规模超过36万亿美元，赤字率已经完全脱离了传统经济学的警戒线。疫情期间，美联储买下了数万亿美元国债，这种"财政部发债，美联储接盘"的模式，正是经典的MMT操作。
在中国，情况更加微妙。官方层面，中国对MMT的态度是高度警惕甚至批判的。中国主流经济学家（如余永定）和央行官员多次强调，要珍惜正常的货币政策空间，坚决不能搞"财政赤字货币化"。但到了2024至2025年，由于经济现实的压力，中国也开始尝试一些"类MMT"的操作。
在中国，央行在一级市场直接购买国债是违法的。所以实际的操作方式是：财政部发行国债，指定工农中建等国有商业银行买入（一级市场认购），再倒手给央行（二级市场买入）。虽然多了一道手续，但实质上还是打通了"央行印钞"和"财政花钱"之间的通道。与此同时，中国开始发行超长期特别国债，期限长达三十年甚至五十年。因为期限极长且由国家信用背书，这种债几乎不用考虑偿还问题，性质上越来越接近于变相的货币发行。
事实上，中国早就拥有一台隐蔽的"定向印钞机"：抵押补充贷款。央行把基础货币借给国家开发银行等政策性银行，国开行再通过"棚改货币化安置"把钱直接发到拆迁户手中。2016年至2018年三四线城市的房价大涨，很大程度上正是这台印钞机推动的。号称"央行请客，地方买单"。
中国和美国的区别是，美国搞得比较露骨，中国搞得比较含蓄。中国没有让央行直接买断国债，但一直在搞"功能性的MMT"，即利用央行的资产负债表，为基建、棚改、化债等国家战略提供低成本的资金支持。之所以中国的操作看起来不像印钞，是因为中国政府独特的“国有银行体系”，可以命令商业银行去买债，而不必央行亲自下场。中间多了一道商业银行的缓冲，使得操作看起来更像正常的信贷扩张，而非赤裸裸的财政赤字货币化。虽然中国政府嘴上批判西方乱印钞，但在面临通缩和债务压力时，身体却非常诚实。
理想和现实
如果印钞是饮鸩止渴，那么正确的应对方式是什么？答案在经济学教科书里写得清清楚楚：结构性改革，允许违约和破产，让市场出清。
如果把经济危机比作一个长期酗酒导致的肝硬化病人，MMT是给他打吗啡让他感觉不到痛，而正确的应对则是做手术切除坏死组织，并强制戒酒。
具体到经济操作上，这意味着允许违约和破产，让市场出清。政府不再兜底，让那些投资失败的房地产商倒闭，让那些借钱盲目扩张的地方融资平台破产。虽然这会带来巨大的阵痛，包括烂尾楼和银行坏账，但这能把被锁死的土地、资金和劳动力释放出来，重新分配给高效的企业。熊彼特称之为"创造性破坏"。
其次是痛苦的去杠杆。不是通过印钱来稀释债务，而是通过紧缩来偿还债务，这通常意味着变卖资产、削减开支、降低生活水平。
最后是打破垄断与减少干预。砍掉对低效国企或特定行业的补贴，把市场准入放开，让民营经济在残酷的竞争中优胜劣汰。
事实上这样的改革不是没有先例。1990年代末朱镕基主导的国企改革就是典型的成功案例。当时大量国企关停并转，数千万工人下岗，那是极度痛苦的几年，但正是这种刮骨疗毒，为中国后来长达十几年的经济腾飞打下了基础。
那么，既然这种“结构性改革”能带来长期的经济腾飞，为什么现在的政府，无论东西方，都越来越倾向于用MMT（印钞）来拖延时间？
原因主要有三个，每一个都非常现实且残酷：
首先是经济需求与政治需求的根本冲突。经济需求是先经历三到五年的剧烈萧条，然后迎来长期繁荣；但政治需求只关心任期内的稳定。如果一位领导人选择刺破泡沫，他很可能在经济好转之前就因社会动荡而下台。正如凯恩斯那句名言："从长远来看，我们都死了。"现在的决策者不仅不想死，还想连任。
其次是"大而不能倒"。今天的经济体比1929年或1990年复杂得多。在中国，房地产与地方债、银行系统深度绑定。如果真让恒大、碧桂园彻底按市场规则清算，可能会引发连锁反应，导致数十家银行倒闭，数百万家庭资产归零。这种系统性风险是任何决策者都不敢轻易尝试的。MMT提供了一种诱惑：只要印钱把洞补上，就能延缓这场雪崩。
最后是现代社会对痛苦的忍耐力极低。二战后的几代人习惯了经济增长和社会保障。在十九世纪，经济危机时工人工资减半可能只引发局部罢工；但在今天，如果全社会收入下降百分之二十，可能会引发剧烈的政治动荡。为了维持稳定，政府倾向于用通胀这种隐形税收来替代显性的破产和失业，因为通胀是温水煮青蛙，而失业是把青蛙直接扔进开水。
归根结底，MMT迎合了当权者希望不受预算约束、权利无限扩张的本能。需要注意的是，由于中国缺乏约束机制，如果MMT在中国全面落地，它造成的后果可能比在美国更严重。
MMT就像经济学上的兴奋剂，它告诉国家：你不必节食，不必运动，只要吃这个药片就能生龙活虎。着完美的迎合了人性的弱点，很少有人能拒绝这种诱惑。西方真正的好东西——对个体尊严的尊重、法治精神、自发秩序，往往很难被移植。反而是简单粗暴、看似科学、便于管理和控制的部分，最容易被权力吸收，发扬光大。

激进的平等主义原本是西方对资本主义的一种批判，后来逐渐演变成了温和的社会民主主义。然而，当它进入中国，被一种渴望快速现代化的急切心态所吸收，就变成了要彻底推翻一切旧秩序的暴力工具，在二十世纪五六十年代彻底摧毁了中国的经济。
1968年埃利希的《人口爆炸》当时引发了对人口过剩的恐慌。在美国，他的理论引发了一些环保运动。但在中国，他的理论与强大的行政力量结合，变成了强制性的独生子女政策。中国当时计划生育理论架构者宋健，在七十年代末深受西方"罗马俱乐部"以及控制论的影响。他们相信，政府可以像控制导弹轨迹一样精确计算和控制人口数量。
