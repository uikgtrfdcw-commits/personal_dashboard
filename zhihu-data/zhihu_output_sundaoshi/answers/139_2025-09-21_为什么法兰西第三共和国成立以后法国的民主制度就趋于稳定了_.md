# 为什么法兰西第三共和国成立以后法国的民主制度就趋于稳定了？

- 日期: 2025-09-21
- 赞同: 0

简单来说，法兰西第三共和国的稳定，并非源于什么完美的设计，而更像一个意外。它是在各方势力走投无路的情况下，被迫妥协的产物。但正是这个谁都看不上眼的权宜之计，让法国大革命中最为稀缺的品质——妥协，成为了社会共识。
要理解第三共和国为何能稳定，首先要明白它之前的法国有多么不稳定。要看清这一点，我们不妨将时钟拨回到第三共和国诞生的前夜。
1846年，法国正处在“七月王朝”时期，政坛腐败，社会矛盾日益尖锐，统治阶级却普遍沉浸在“天下太平”的幻觉中。当时担任议员的托克维尔，敏锐地察觉到民众中蕴藏的巨大不满。他在议会发表演讲，向那些麻木不仁的政客们发出了警告。他演讲中最著名的一段是：
 “……人们说，没有什么危险，因为没有发生暴动；人们说，因为社会表面没有可见的动荡，革命就离我们很远。先生们，请允许我告诉你们，我认为你们错了。……​我们正睡在一座火山口上。​”
他的警告被当成了危言耸听。然而，仅仅三周后，革命的火焰就在巴黎街头被点燃，七月王朝轰然倒塌。
托克维尔之所以能看得如此透彻，不仅因为他是洞察现实的政治家，更因为他是一位卓越的历史学家。他生前最后一部著作《旧制度与大革命》出版于1856年，三年后他便与世长辞。他写作此书时，法国刚刚经历了1848年革命和拿破仑三世建立“第二帝国”的新一轮动荡。这让他深感困惑：为什么大革命后的几十年间，法国始终在“革命”与“专制”之间来回摇摆，无法建立一个稳定的政治制度？
为了回答这个问题，他才决定深入研究大革命前的“旧制度”，试图从历史的深层根源中寻找答案。这本书实际上只是他宏大写作计划的第一卷，他本想接着写第二卷来分析大革命本身和拿破仑帝国的，可惜未能完成便与世长辞。
事实证明，法国的“政治学徒期”比他预想的还要漫长。从1789年到1870年，这个国家像钟摆一样来回摇摆，在君主制、共和制和帝国之间循环往复。人们一次次满怀希望，却一次次跌入失望。大革命理想未能化为共和制度，反而让拿破仑独裁登场；复辟王朝看似恢复秩序，却因守旧而再度被推翻；七月王朝代表大资产阶级，却依然排斥大众，最终自食恶果；第二共和国带来了普选权，
但当时法国公民社会尚未成熟，观念的水位仍然停留在对政治强人的幻想上
，于是路易·波拿巴顺势登基，并在民众的默许下发起政变，黄袍加身。
到1870年为止，法国在不到一百年的时间里，先后经历了两次君主制、两次共和制和两次帝国。各种体制轮番登场，却没有一个能弥合深层的社会分裂。君主派、共和派和帝国派三足鼎立，谁也无法彻底压服对方，国家在一轮又一轮的希望与挫败中反复挣扎。
真正的转折点发生在1870年。普法战争的惨败让第二帝国土崩瓦解，法国被迫在废墟上重新寻找出路。在一片混乱中，第三共和国仓促诞生。起初，几乎没人看好它，国民议会里君主派甚至占多数，许多人只把它看作一个短暂的过渡。但偏偏就是这个“临时拼凑”的政体，顽强地存续了七十年，成为法国历史上第一个长寿而稳定的共和国。
它的成功带有几分偶然。因为各派谁也推不出一个公认的国王，共和制成了分歧最小的选择。这个制度权力分散，内阁频繁更替，政府看似脆弱，却恰好避免了任何强人独揽大权，政治人物只能在无尽的妥协中前行。与此同时，共和派将精力投向广袤的农村，承诺保护农民的土地所有权，让他们相信共和国才是自身利益最可靠的守护者，从而赢得了这个国家最庞大的社会群体的支持。
其实，第三共和国也曾面临危机——布朗热事件、德雷福斯事件差点掀翻政权，但这一次法国人没有再依赖政变或街头暴力，而是通过法律、议会和舆论的博弈来和平解决。每一次危机爆发都必然伴随着利益冲突，但每一次冲突的化解，都增强了民众对共和制度的信任。
于是，一个原本谁都不看好的“权宜之计”，居然靠危机教育与漫长妥协熬成了法国历史上第一个真正稳定的政体。它没有完美的设计，没有救世主式的领袖，却在反复摇摆与修修补补中存活下来，并最终带来前所未有的稳定。
然而，故事并未就此结束。1940年，第三共和国在纳粹德国的闪电战中羞辱地崩溃。战后重建的第四共和国，几乎照搬了第三共和国的议会制度，同时也继承了其政治不稳的顽疾，内阁如走马灯般更迭，无力应对阿尔及利亚战争等殖民地危机，国家再度濒临瘫痪。
正是在这样的危急关头，1958年，戴高乐再度出山，主持制定了新宪法，开启了法兰西第五共和国。新宪法一改过去议会独大的传统，建立起“半总统制”，赋予民选总统强大的行政权力，尤其是在外交和国防领域。这既保证了国家的决策效率和稳定，又通过议会和司法构成制衡，避免了个人独裁。
需要强调的是，戴高乐的集权改革，与波旁王朝的君主专制及拿破仑的军事独裁有着本质区别。波旁王朝依赖的是早已过时的“君权神授”，拿破仑依靠的是个人魅力与赫赫战功，两者都缺乏持久的制度根基。而戴高乐的改革，则是将强大的总统权力“制度化”，把权力关进现代宪政的笼子里，其合法性来源于全民选举，并受到明确的任期制衡。
从君权到帝国，再到共和国，法国的政治历程几乎是一部“试错史”。波旁王朝的回头路走不通，拿破仑的强人政治只能昙花一现，第三共和国教会了法国人妥协的价值，第四共和国暴露了议会政治的缺陷，而第五共和国则把强力领导与宪法框架结合起来，找到了一种新的平衡。法国的百年动荡，最终换来的是一个既能容纳多元意见，又能维持制度稳定的政治秩序。
法兰西的百年求索，早已超越一国一地的历史，它变成了一个关于现代政治的深刻寓言：如何在赋予人民权利的同时，避免“多数人暴政”？如何在建立强大国家的同时，防止权力异化为专制？法国用一部惨痛的试错史给出了自己的答案：没有一劳永逸的终点，只有在自由与秩序的持续博弈中，不断寻求脆弱的平衡。这或许是所有现代国家都无法回避的课题。
