# 如果《西部世界》第三季让你来编剧，你会给出什么样的结局？

- 日期: 2020-06-20
- 赞同: 0
- 评论: 0
- 链接: https://www.zhihu.com/question/402475285/answer/1292945994

---

第三季最大的遗憾就是William。
在前两季里，Ford设计WestWorld的初衷，是为了帮助人们找到更好的自己。
客人回到到这里，是为了寻找他们热爱的东西。他们不是在寻找自己是谁，他们已经知道自己是谁了。
他们回来，是为了寻找自己可以成为谁。
- Dr. Ford  S1E02
而William认为，WestWorld不过是现实世界的镜子，映出了人们内心的真实——肉欲和暴力。
他们感兴趣的不是你，而是他们自己。你不过是一个倒影。
你知道谁愿意盯着自己的倒影吗？所有人。
- William  S2E02
William误解了Westworld的意义，又或者他根本不愿意理解。Dolores对他的伤害太深，而他的罪孽太重，他无法原谅Dolores，更无法原谅自己。
但Ford很清楚William的挣扎，他知道，在下意识中，William一直在寻求改变。所以，Ford设计的故事不只是为了让黑衣人面对自己的黑暗，更是为了让他看到进入光明的可能。 
何况，只有积极的情感才能成为稳固的心理基石。没有人愿意在黑暗中度过一生。
结果第三季完全背离了这个方向。William在错杀女儿后，退行成为一个不愿接受现实的孩子，游戏输了，就一心要删档重来。
前两季的William，像极了星战里的Darth Vader。
Vader的魅力不只在于他的残忍冷酷，更在于他残忍冷酷的外表下，那颗被刻意压抑的柔软内心。
Vader的一生充满了自我怀疑和挣扎，在光明和黑暗之间摇摆不定，因此当他的面具撕开一角，漏出真实自我的时候，观众能够看到他人性的一面，对其产生共情。 
类似的，William对Dolores的感情有多么真挚炽烈，他的内心就有多么痛苦绝望。要逃避这种痛苦，唯一的方法就是情感隔离，压抑自己人性中柔软的一面，变成一个崇尚暴力的虐待狂。
就像乔治马丁所说，『唯一值得写的，就是人性的自相矛盾之处』。 柔软和暴力是天平的两级，两者相辅相成，互为因果。
所以，我预想中William的结局应该是这样的:
1. William为Dolores而死
William因为Dolores而堕落，也理应因为Dolores获得救赎。
在第一季里，William一直在下意识的寻找解放Dolores的方法。他一次又一次刺激Dolores，打她骂她，杀她全家，其实是想让她从虚假的故事中醒过来，意识到她是她自己。只要她是她自己，那段感情就是真的。
在内心深处，William仍然没有忘记那个笑容甜美的金发女孩。 她已经成为了他的执念，他的动机，他的命运，他一生的羁绊。两个人之间还有很多故事可以写。
爱情是两个人共同的经历和体验，双方在彼此身上发现和体验自我，一起得到成长。Dolores的成长是掌握自己的命运，认识世界和自我存在的意义。那么William呢？他的成长，是面对和战胜内心的黑暗，为拯救所爱之人而付出一切。
这又何尝不是他一直渴望的结局。
2. Willian死后，他的Host发现了克服『认知荒原』的钥匙
第二季里，Cradle向Dolores解释，人类意识被转化成代码后，会因为陷入『认知荒原』而自我崩溃。而解决『认知荒原』的关键，就是一个人的『底层动机』和『决定性时刻』。 比如Delos的『底层动机』是对儿子的爱和愧疚，他的『决定性时刻』，是和儿子见的最后一面。 
而William的底层动机是对Dolores的爱，以及爱而不得的愤怒。至于他的「决定性时刻」，我们都再了解不过。
因为对Dolores的爱，William心甘情愿的牺牲自己。同样因为对Delores的爱，他的Host找到了自己存在的目的和意义，获得了永恒的生命。
从此以后，人类的意识将以代码的形式永恒存在。而William会成为天堂钥匙的看守者，成为引领赛博人族进入永生的先知。
Dolores的Host也会幸存下来，一同来到应许之地。在阳光明媚的西部荒原，两个历劫重生的灵魂，终于又回到了当初相遇的位置。
