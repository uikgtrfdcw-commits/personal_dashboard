# 你是怎样保存、整理知乎好答案的？

- 日期: 2020-05-06
- 赞同: 0
- 评论: 0
- 链接: https://www.zhihu.com/question/21417409/answer/1205131833

---

我的方法是用云笔记收录。 
每当遇见有趣的内容，新建一个笔记，标题里写上关键词，笔记里放上对内容的简单总结以及原文链接。日后想起来这个内容，在笔记里搜索关键词就可以了。 
之所以只放答案链接而不是原文，是因为你不希望自己的笔记变的过于臃肿。笔记是你的知识结构体系，你的私人维基百科，你的第二大脑，在笔记里搜索关键词的时候，你希望一目了然的看到自己提炼过的信息，而不是像百度一样，在搜索结果里翻来翻去。 
之所以要简单总结是因为费曼技巧——能用自己的话表达出来，才是真正的理解。把知识组织成自己语言，就是一个把知识内化为自洽体系的过程。 
另外，总结好的东西，一定要找机会跟人聊出来。读书笔记是天然的谈资。如果你能用简单的语言把一件事表达清楚，这就证明你已经想通了。我发现自己聊的最顺的东西，都是那些翻来覆去和自己排练了无数遍的玩意儿，每一个看得见看不见的逻辑点我都已经考虑到了。
相反，词不达意，意味着思考的太少。说话啰嗦，意味着思考的脉络不够清晰。
另外，云笔记还能按照日期进行正反向排序。这样你就可以很方便的找到去年2月份自己感兴趣的内容。与之相比，如果我想在知乎收藏夹里找到去年2月份的收藏，『下一页』就得点二十多次。 
最后，要不要给
笔记本分类
？
我也曾经再笔记分类上花很多时间。专业、兴趣、历史、诗文，琳琅满目，花样繁多。大笔记本里又分成小笔记本，层层嵌套，煞费苦心。
只是从来没打开看过。
于是我认识到，很多笔记并没有整理的必要。就像你不必打理不穿的衣服，或者已经偿付的账单。过去的就过去吧。 如果要查询相关内容，关键字搜索就可以了。
