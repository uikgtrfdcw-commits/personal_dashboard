# 什么是正反馈循环？

- 日期: 2025-12-28
- 赞同: 0
- 评论: 0
- 链接: https://www.zhihu.com/question/1988655666584387895/answer/1988655785761317472

---

想象一个最常见的场景：话筒啸叫。
正常情况下，你对着话筒小声说，音箱就小声播放；你大声喊，音箱就大声播放。投入与产出成正比，一切都温和而可预测。
但当你把话筒对准音箱，魔法就发生了。一个微弱的声音从音箱发出，被话筒捕捉、放大，再更响亮地从音箱播放出来，然后又被话筒捕捉、再次放大……这个瞬间形成的循环，让一个微不足道的初始噪音，很快变成了一声震耳欲聋的尖啸。
这就是正反馈循环：系统的输出变成了输入的一部分，不断自我强化。在这里，一分耕耘不再等于一分收获，而可能是一千分、一万分的收获。它还有很多别名：链式反应、非线性增长、蝴蝶效应、左脚踩右脚、滚雪球、病毒传播。
一旦理解了这个机制，你会发现它在现实中无处不在。
想想股市泡沫是怎么形成的。你的同事告诉你他买了某只股票赚了钱，你心动了，于是也买入；聚会时你把这个消息告诉朋友，他们也跟着买入；越来越多人买入推高了股价，而上涨的股价本身又成了"这只股票值得买"的证据，吸引更多人入场。价格就这样一圈一圈地自我强化，直到某一天，有人开始获利了结，同样的循环反向启动，踩踏式的崩盘随之而来。
社交媒体把这种效应放大到了极致。在传统媒体时代，一个热点的形成可能需要几天甚至几周；而在社交网络上，短短几个小时，一个原本无人问津的小事就能演变成全网热议的焦点——当然，它被遗忘的速度也同样快得惊人。
那么，正反馈循环为什么会出现？根本原因是系统内部的元素相互关联，而非彼此独立。话筒啸叫之所以发生，是因为话筒和音箱在物理上被放到了相互影响的位置——如果它们隔着两堵墙，输出就不可能变成输入。
举一个类似的例子。比如你经营一家线下咖啡馆，每日的顾客数量是可预测的。因为每个顾客是独立决策的，他们之间互不影响。有的人特别喜欢咖啡，有的人又特别讨厌，极端情况会相互抵消，最终客流量总是在平均值附近波动（中心极限定理）。你可以准确预测下个月的客流量并作好准备。
但网红咖啡馆就不同了。顾客的决策是相互影响的。一旦有人在社交媒体上打卡，就会吸引更多人来拍照打卡，更多的打卡又带来更高的曝光。这会导致资源高度集中：赢家通吃，输家颗粒无收。（也就是统计学上的幂律分布）同一条街上，口味差不多的十家咖啡馆，往往一家挤破头、九家门可罗雀。
理解了这一点，也就能理解为什么近年政策常常给人"翻大饼"的感觉。当市场中缺少足够多独立分析、独立决策的主体时（负反馈调节机制），企业和投资者很容易相互观望、相互跟风。某个行业刚露出一点政策利好的苗头，大家一窝蜂涌入，产能迅速过剩；风向一变，又一窝蜂撤退，留下一地鸡毛。表面上看是政策在左右摇摆，实际上是市场本身的正反馈循环把任何信号都放大成了极端反应，逼得政策不得不反复纠偏。
在这种由正反馈循环主导的世界里，“蝴蝶效应”是常态。一个微小的、几乎无法察觉的初始优势——可能只是一次偶然的利好因素，一个幸运的时间点，都可能在正反馈循环的放大下，瞬间变成决定成败的关键因素。你无法预测下一首“神曲”会是什么，因为引爆流行的，往往只是某个主流媒体的随机转发。你也无法预测下一个塌方的明星是谁，因为一句无心之失，就可能导致炙手可热的大V一夜变凉。
这为我们提供了一种新的世界观。它告诉我们，许多戏剧性的结果，最初往往只是蝴蝶煽动了一下翅膀，经过系统内部那套看不见的、相互关联的、不断自我强化的作用机制导致的。认识到这一点，或许可以帮我们减少一些对精确预测的执念，转而开始去欣赏和敬畏那些意外的复杂之美。
注：
中文语境中，正反馈循环（positive feedback loop）容易被误解为"良性循环"（virtuous cycle），但前者本身是中性的——既能放大好事，也能放大坏事。与之相对的概念是负反馈循环（negative feedback loop），指系统自我纠偏、趋向稳定的机制。
此外，一些心理学译者会将其与“正向激励”（positive reinforcement）混同。但后者是指心理学上的自我奖励。
