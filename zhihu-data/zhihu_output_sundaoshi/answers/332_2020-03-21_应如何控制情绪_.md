# 应如何控制情绪？

- 日期: 2020-03-21
- 赞同: 0
- 评论: 0
- 链接: https://www.zhihu.com/question/53299143/answer/1094464765

---

当我们说『控制』情绪的时候，我们的意思其实是『压抑』情绪。
这实际上是不可能完成的任务。情绪是你内心的最真实的感受，你不能永远压抑自己。
情绪就像生命，有自己的逻辑、感受和意志。如果你逃避、敌视和压抑它，它会变成暴戾和有毒的东西。但如果你尊重、理解和接纳它，它就会变成心灵的养料和成长的契机。
情绪是个美妙的自我认识的机会。
强烈的正面情绪意味着过去某种积极的情感体验。强烈的负面情绪意味着过去的某种未曾解决的心理创伤。那些埋藏在内心深处的种种底层认知，一到合适的时机，就会以情绪的形式展现出来。
贴一段Tim Ferris和Dr. Gabor Mate非常有意思的对话：
给我讲一段让你不爽的经历。
我雇人做一件事，结果回来发现他没做到。
你当时的情绪是？
愤怒。
好的。问一个愚蠢的问题：你为什么感到愤怒？
这个家伙承诺我一件事，但是他没做到。
是的，他没做到。这和你有什么关系？你为什么感到愤怒？
他不尊重我，不关心我的感受。
好的他对你不尊重不关心。什么样的人得不到尊重和关心呢？
呃...不值得被尊重和关心的人。
再问一个问题，他没做到有没有可能是因为别的什么原因？
好吧...他可能病了，可能家里有事，可能弄错了时间…我自动就相信了最坏的情况。
对的。
我们相信的不是事实，而是自己对事实的理解。
这是佛陀说的。
现在的问题是，你为什么会这样理解。
最后一个问题，这是你第一次感到不值得被尊重吗？
可能…不是。
通常都不是。激烈的负面情绪往往要追溯到很久以前的心理创伤。
我们相信的东西并不来自于现在，而来自于过去。
反过来讲，如果你坚信自己是世界上最聪明、最正直、最值得尊敬的人。有个家伙承诺了你一件事，他没有做到，你认为是什么原因？
任何一种可能...反正不是因为不尊敬我...
现在我问你，之前真正让你感到不被尊敬不被关心的是谁？
...是我自己。我心中仍然有一部分相信自己不值得被尊敬。
你看，负面情绪其实是个美妙的认识自我的机会。我们所有的负面情绪，都反映了我们无法克制的某种心结。
