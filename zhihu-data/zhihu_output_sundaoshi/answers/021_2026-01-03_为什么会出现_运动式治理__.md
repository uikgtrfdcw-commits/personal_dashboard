# 为什么会出现“运动式治理”？

- 日期: 2026-01-03
- 赞同: 0
- 评论: 0
- 链接: https://www.zhihu.com/question/1990847167808701734/answer/1990848683219108246

---

运动式治理的根源在于常规治理机制的失灵。当日常执行机制无法有效解决积累的问题时，就只能通过集中动员、突击整治来应对。
而失灵的根源在于激励机制的扭曲：事情做成了没有特别的奖励，但做错了一定会被问责，风险远大于收益，因此多一事不如少一事。人人都知道问题在哪，但眼不见为净，导致问题逐渐累积。
当问题最终爆雷，高层震怒，就会启动第二阶段：运动式治理。
一旦进入运动阶段，原本的“躺平”倾向就会跳跃成“过度执行”倾向。一旦有了明确上级指示，考核指标就从平时的“不出乱子”变成了“干出成果”。
这种情况下，如果执行力度偏大，无非就是“工作方法简单粗暴”，属于能力问题。而如果执行力度偏小，那就是“消极执行中央政策”，属于态度问题，意味着政治前途不保。因此每一级官员为了给自己留出“安全冗余”，宁左勿右，都会在上级指令的基础上加码 10%。当 5-6 层都这么操作（省-市-县-乡镇-村/街道），到基层时政策已经完全变形。
且官员行为的唯一评价标准是“上级满不满意”。上级精力有限，不可能看到所有细节，只能看到“态度”，这导致了“表演式执行”，大量资源被投入到“做给上级看”的活动中——开会、汇报、造材料、搞形式。这使得运动式治理变得极其低效和昂贵，基层苦不堪言。
同时，官员为了避免被问责，倾向于报喜不报忧。政策导致的负面后果、群众的真实反应、运动的实际成本，这些信息在向上传递的过程中会被逐级美化。每一级都在向上级报告“坚决贯彻、成效显著”，最终的结果就是上面政策不切实际、下面苦不堪言。这导致决策层长期处于信息茧房中，看到的都是“形势一片大好”，感受不到基层和社会的真实压力。或者即使意识到问题的存在，为了维护权威，也会坚持到不能再坚持。
等到问题终于以某种无法掩盖的方式爆发出来（比如群体性事件、经济数据断崖式下跌、舆情失控），就会突然急刹车，迅速淡化处理，一切回归如常，仿佛什么都没发生过。直到下一次运动，周而复始。
运动式治理本质上是在透支未来。每次运动都会消耗基层的执行能力和社会的配合意愿，下一次动员就会更困难。且运动的不可预测性极大提高了社会运行成本——企业不敢长期投资，民众不敢做长远规划，整个社会都处于“等靴子落地”的焦虑中。
要打破这个循环，需要的不是更猛烈的运动，而是深层的制度变革。但这是另一个更复杂的话题了。
