# 蒋介石为什么在大陆土改失败？

- 日期: 2020-05-05
- 赞同: 0
- 评论: 0
- 链接: https://www.zhihu.com/question/342452591/answer/1202532718

---

和一般看法不同，蒋介石对土地问题其实相当重视。土地问题是国家经济制度建设的根本问题，再说『平均地权』和『耕者有其田』本来就是国民党『民生主义』的政治纲领。只是相对于共产党积极的土地政策，蒋的改革手腕要温和的多。 
国民政府形式上统一中国后，当年就由胡汉民草拟了『土地法』，在部分省份试行，这就是『二五减租』政策。但政策的推行遇到了地方既得利益者的极大阻力。比如浙江地方官绅联名上书：『浙江单行二五减租办法，行之数载，弊端百出』，提出减租运动与私权制度不兼容，请政府命令取消。面对愈演愈烈的地方矛盾，蒋介石不得不致电浙江省主席鲁涤平： 
浙江二五减租，农民未受其益，社会多蒙其害，且徒增纠纷。于全国未有确定规章通令以前，不如暂时用明令取消，即以此项租税作为江浙国防之可用也。
对于政策推行的失败，蒋介石认真做出了反省。他邀请财经专家、各级官员汇报情况，共同研究，从而更加翔实的了解到减租政策在地方实施过程中遭遇的具体困难。比如南开大学教授何廉曾向蒋介石反映，浙江大部分地区实行的是一种特殊的佃户制度，即大地主把土地租给大佃户，大佃户又把土地租给小佃户，而减租政策只顾及了后一部分。
经过和各方沟通，蒋意识到，二五减租过于理想化，不足以应付基层社会的复杂情况，自己需要拿出一个更加细致、更加符合实际的方案。
国民政府第二次推行的土地改革政策，是由留德地政学家萧铮制定，由汉口『剿匪』总部颁布的『剿匪区内各省农村土地处理条例』。一方面，『条例』承认土地私有,『以确定所有权为原则』。另一方面，『条例』规定业主拥有土地不得高过200亩，超额部分征收所得税。同时地方省政府成立农村复兴委员会，将乡镇共有的荒地按照人头分给农户。此外『条例』还在减租、借贷方面做出了详细规定。 
也就是说，蒋此时采取的是一种『渐进式』的改革方案，通过兼顾地主和农民双方的利益，维护社会基层稳定。 
然而，这种缓和式的土地改良办法，极难在农村基层社会真正得到落实。由于各地乡绅阶层的基层势力盘根错节，以及国民党『四一二』清党后基层组织孱弱，导致国民政府的权威触角很难深入广袤的农村地区，更别说推行这样一个牵涉广泛利益的土地查报政策了。一年后，仍未看到改革成效的蒋介石亲自出面向地方官员施压 ：
各省县农地之查报日期再重申，电令催报…如再不从，则不问何人，一体严惩，必罪无赦。
蒋的改革政策不但在江浙腹地推行不顺利，在西南地区也遭到了地方实力派的抵制。而抵制的理由竟是嫌改革不够激进。在西南反共前线，共产党的土改政策给地方军阀带来巨大压力，地方军阀更有意愿和动力，去采取比较激进的改革措施。比如十九路军在闽西地区施行『计口分田』，引起了政界和社会上的广泛反响，『多赞成闽西之办法』。阎锡山在山西提出了一套『土地归村公有』的改革方案，也和蒋针锋相对。不得已之下，蒋最终表示部分妥协：
对土地问题，在江西主缓进，不主急进，以改良的办法，逐渐进到解决土地问题的目的。阎主任内忧外患，要想法解决，主张解决土地问题，此事在山西是可做的。中国如此之大 ，不特各省制度难求统一，即法律亦各省不能一律，因为各省人情风俗地理气候种种均不相同，处处都影响到经济问题，所以制度法律难求划一 。
换句话说，蒋介石被迫开了一个特例，允许阎锡山在山西进行『急进』的土地改革。但江西和其他省区，则必须坚持原有的土改办法。不久之后，在蒋的授意之下，萧铮在国民党第五届全国代表大会上提出『积极推行本党土地政策案』，并设立了土地专门委员会，专门贯彻实行蒋介石所信奉的渐进式土地改革。
然而正当蒋准备有大一番作为之时，抗战全面爆发。蒋的土改方案也就不了了之了。
最后，站在蒋的立场上看，他在试图解决土地问题时，主要面临着三方面的约束： 
其一，面对共产党『打土豪分田地』的政治主张，蒋必须找出令人信服的解决方案，但又不能照抄共产党的套路。 
其二，身处『剿匪』前线的川陕地区饱受袭扰之苦，经济利益受到严重威胁，因此对改革并不排斥。但对于中央控制的广大国统区，红色势力已经基本消灭干净，对改革的接受力度也要差一些。 
其三，蒋不但要应付国民党内部的各种分裂势力，还要应对日本帝国主义侵略、国家财政危机、地方吏治废弛等诸多方面的现实压力，在这些压力之下，蒋所试图解决的土地问题已经不再是单纯的土地问题，而是更深层次的资源调配问题。 
总之，蒋在土地改革方面所面临的现实困难和政治包袱，要远远大于共产党以及阎锡山这样的地方实力派。因此最终改革成果有限，也是心有余而力不足。
