# 如何评价游戏《刺客信条：起源》？

- 日期: 2020-05-01
- 赞同: 0
- 评论: 0
- 链接: https://www.zhihu.com/question/59901707/answer/1192731471

---

三个意见：
1.  缺少一个邪恶冷酷的反派角色镇场子。
凯撒是个没见过世面的老头，克里奥佩特拉看起来只对Party感兴趣。金甲虫倒是有点意思，心狠手辣，笑里藏刀。可惜这厮血条太短，巴耶克稍微一发力，他就去见阿蒙神了。导致我一度希望他是假死，大结局再跳出来给我个惊喜。
当然育碧不会塑造人物是老问题了。系列上一个让人印象深刻的大BOSS是谁？凯撒.波吉亚？
2. 故事情节聊胜于无。巴耶克的复仇确实令人信服 ，但其他的什么远古秩序守护者啊，自由和控制啊，Abstergo的阴谋啊，味同嚼蜡，如同儿戏。连凯撒被刺这种桥段都能够拍的平平无奇，导演是个人才。
刺客信条前两部是扎实的硬科幻，对历史，对宗教，对人类起源和未来进行了颠覆性的解读。编剧Corey May明显有丹布朗的影子。可惜系列后续情节还是落入俗套。
3. 好吧我痛恨沙盘游戏，尤其是那些堆成小山的支线任务。
平庸的任务不如没有任务，浪费时间就是谋杀生命。每次在地图上重复赶路，我都会无比怀念魔兽三里精妙的关卡设计，使命召唤四里精确到每一秒钟的舞台效果。 年纪大了看这些动不动100小时+的沙盘游戏，也真是没啥胃口。五个字：能量密度太低。
