# 什么是幽默？什么是高级幽默？

- 日期: 2020-06-22
- 赞同: 0
- 评论: 0
- 链接: https://www.zhihu.com/question/22529282/answer/1296484956

---

幽默的组成元素有二：出人意料，不谋而合。一来是给你介绍了新玩意儿，让你耳目一新；二来和你看法一致，让你自我感觉良好。
前者我们称之为有趣，后者我们称之为投缘，如果既有趣又投缘，我们便称之为幽默。
幽默是一种绕有新意的奉承，一种别具匠心的精神按摩。幽默让我们愕然不语，会心一笑。幽默让我们智识上有所收获，同时自我认同上又得以增强。
缺乏同理心的人不懂幽默，因为他们的注意力都在自己身上。没有主见的人也不懂幽默，因为他们的想法都在其他人的意料之中。
一个幽默的人，既要对世界怀有好奇心，又要有足够的安全感。既要把问题看得清楚，又要保持足够的谦卑。
你无法一夜之间变得幽默。幽默就像魔术，是一门需要长期打磨的手艺。很多看似灵光一现的聪明话，背后往往是多年的思考和观察。
