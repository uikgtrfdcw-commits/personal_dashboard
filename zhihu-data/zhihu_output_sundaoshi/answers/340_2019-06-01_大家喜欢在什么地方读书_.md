# 大家喜欢在什么地方读书？

- 赞同: 0
- 评论: 0

---

我特别喜欢在咖啡馆读书。
尤其在那种不是特别热闹，但也绝不冷清，神闲气定，视角宽阔的小角落，可以清晰的听到时光的流淌。
没有太多人关注，又没有被世界遗忘，有点紧张但又不至于分心，安静的做自己喜欢的事情，大脑迅速进入状态，感觉特别好。
也尝试过在家里看书，在单位看书，在地铁里看书，在公园看书，但都没有这种若即若离，炫耀而又矜持，渴望关注而又保持距离的分寸感。
所以我想，读书也是同样的道理。既不能独学无友，亦不可让人言淹没了初衷。随和却偏执，虚荣也淡泊，一张一弛，文武之道也。
既有自己的节奏，又不脱离实际。既保持独处的空间，又与人心意相通。托身于市井，不拘于世俗，扎根于尘世，超脱于外物。雷鸣电掣，秋霜春露，风拂自在，枝干轻舒。一切都刚刚好。
Caffe Bene
从前常去的咖啡馆就在单位对面，一家购物商场楼上电梯旁边。如果坐到好的位置，一眼望去，几层楼的商铺人流尽收眼底，视野特别棒。
我以前一直以为，当你盯着电脑看了一整天后，没有什么能比坐下来边喝咖啡边放松睫状肌，更能让人心身愉悦的了。而现在我的看法更进一步：空旷的视野不但有益于身心健康，还能让你变得更聪明一些。
这有点像跑步机。它给你一个稳定可预期的感官反馈，让你的头脑和感官恢复链接。而一旦你对身体的感知足够敏锐，你就可以进一步觉察到情绪的流动 ，甚至触碰到潜意识和本能。这也就进入了所谓的正念状态。
用更好理解的话讲——这有点像在做白日梦。有时候你突然想起了一件有意思的事儿，头脑里惊涛骇浪，拦都拦不住。那正是你头脑中浮现的潜意识。潜意识非常擅长发现事物之间的细微联系，它总是能捕捉到更多信息。
可惜我太晚才意识到这一点——这家咖啡馆9月底就关门了。流浪了两个月后，我悲伤的发现周围并没有类似的地方可去。此处可待成追忆，失去了才知道珍贵，人世间最大的悲哀莫过于此。
视野的力量
周末的早上，阳光从咖啡馆门口的落地窗倾泻而入，意识悄悄的脱离掌控，在窗外的景色中四处流动。我感到久违的平静，又略有所悟。
在家看书的时候，如果哪里想不通，我的第一反应是打开手机，然后思绪就会被嘈杂的信息流淹没。而在咖啡馆的时候，遇到同样的情况，我的第一反应是的注目于窗外，沉浸于当下。
是窗外的视野让我沉静下来。
为什么窗外的视野会有这种效果？想到了几个原因：
1. 生存本能。这是两百万年来人类遗传进化的结果——良好的视野意味着安全的环境，可以放松警惕。
2. 思维热身。就像在NBA比赛里，持续稳定的投篮刺激可以让球员找到手感。思考问题时，持续稳定的视觉输入可以让大脑迅速进入状态。抖腿和吹口哨也有类似效果。
3. 距离感。减轻了无形的环境压力，置身事外，旁观者清，更容易掌握事物的全貌。
4. 意识锚定。托旷观以适性，聊纵目而怡情，好的视野让我们嘈杂的表层思绪得以依附，进入一种类似无意识的出神状态。
我很幸运，在心绪烦躁的时候，总能找到一个视野空旷的角落。我想如果天堂真的存在，那大概就是咖啡馆的样子吧。
