# 努力的界限到底在哪里？

- 赞同: 0

对楼主的体会感同身受。
以前上学的时候超级拼命，功课不好，就反复琢磨课本到深夜，不善交际，就使劲参加各种活动。总之就是不断给自己加压，相信努力就一定有收获。很长的一段时间中，我都在这种打鸡血 - 彻底放弃 - 心情低落 - 打鸡血中循环。做事的效率自然也可想而知。
努力也要恰到好处。这个分寸的把握是有学问的。用力过度，你会感到焦虑。完全躺平，你又会感到无聊。只有和自身承受限度相匹配的努力，才能获得持续稳定的进步。努力踮脚够一下就能碰到是最好的。
当然，认识自己是最难的，人们很难分清自己的边界在哪里。我认为，理解自己的能力边界是成长的重要课题。
当你明确知道自己的能力边界在哪里，努力就应该是一件自然而然的事。它并不应该是某种需要被时常觉知的东西，不应该被夸大、排练或者表演。努力应该是我们生活的一部分，而不是什么挂在生活之上的负担。徐佳鑫说，太努力的人走不远。真正走到最后的人靠的并不是努力，而是格局和喜欢。
…
现在想一想，我们的「努力强迫症」是从哪来的？
一来，我们从小就被灌输世上无难事，耕耘就一定会有收获。可惜生活不是童话，世上当然有难以做到的事，耕耘当然不一定有收获。一个人过于迷信主观努力，遇到大的困难就难以理性面对。 
二来，浮躁的社会氛围也让我讨厌长远规划，只喜欢用盲目低效的努力来弥补。这种短视在今天简直层出不穷。公司要立刻拉动销量，学校要快速提高升学率，政府要年内出政绩。战术的勤奋，背后往往是战略的懒惰。
…
有趣的是，最初认识到这个问题是在健身房。有段时间每天撸铁到腿软，一段时间后效果先不谈，伤病是源源不断。
于是我意识到：一个人每天的训练量存在一个上限。
过度的训练往往会起到反作用，越努力越受伤。
当然，我并不是反对健身运动。事实上刚好相反，我认为科学的的力量训练是一切运动项目的基础。力量训练会影响到肌肉平衡和关节稳定，乃至于改善日常生活中的动作习惯。健身追求的是最佳的生活状态，而不是竞技状态，这比很多追求身体极限的运动要养生的多。
健身的成果是肉眼可见的。你没办法欺骗自己。你没办法速成，没办法依赖别人，只能亲自一个动作一个动作去体会和学习。这不是一件容易的事。一个人养成了规律的运动、作息和营养习惯，一定是在做事方法上拥有某些过人之处。 
好像有点跑题了... anyway，健身代表着一套科学的学习和实践体系，这对解决生活中的实际问题是有帮助的。比如为了练出效果，你需要考虑一大堆变量：营养、睡眠、天赋、环境、伤病、健身伙伴、生活压力、经济条件、主观努力。这其中，主观努力仅仅是影响因素之一，甚至效用非常有限。即使你一切都做到完美，短期内也未必能看到效果。最后还是要耐下心来，循序渐进，不断积累，做好长期规划，徐徐而图之。
毕竟，有些努力绝非一夕之功。
