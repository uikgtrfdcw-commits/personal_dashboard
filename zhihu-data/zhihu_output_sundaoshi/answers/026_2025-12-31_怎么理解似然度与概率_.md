# 怎么理解似然度与概率？

- 日期: 2025-12-31
- 赞同: 0
- 评论: 0
- 链接: https://www.zhihu.com/question/269204682/answer/1989665488612127693

---

从贝叶斯公式的角度来理解，“似然度”可以近似理解为新证据对后验概率的支持力度。
就拿经典的侦探小说的场景来举例：
L怀疑夜神月是凶手，根据他之前收集的信息，这个概率大概是10%。
P(夜神月是凶手) = 10%（先验概率，即最初的判断）
L接近夜神月后，发现夜神月是超高智商。而凶手的作案手段，明显具有高智商特质，于是夜神月是凶手的概率上升为80%左右。
P(夜神月是凶手|高智商) = 80%（后验概率，即看到证据后更新的判断）
而"似然度"，其实就是"高智商"这个新证据对目标推理的支持力度。严格数学表述，就是"如果夜神月是凶手，他是高智商的概率"：
P(高智商|夜神月是凶手) = 95%（似然度）
而整个贝叶斯推理的过程是：
P(夜神月是凶手|高智商) = P(夜神月是凶手) × P(高智商|夜神月是凶手) / P(高智商)
80% = 10% × 95% / 11.875%
其中 P(高智商) 表示在L的嫌疑人范围内，高智商的人占比约11.3%。这个数字可以这样算出来：
P(高智商) = P(高智商|是凶手) × P(是凶手) + P(高智商|不是凶手) × P(不是凶手)
P(高智商) = 95% × 10% + 2% × 90% ≈ 11.875%
（这里假设普通人是高智商的概率只有2%左右）
CPU已经快烧了。我觉得"新证据对目标推理的支持力度"这个简化解释就已经够用了。先建立一个直觉和数学感，如果以后实际工作需要的话，熟练度提升，理解自然会更精确。
注：
其实严格来讲，似然比P(高智商|夜神月是凶手) / P(高智商) = 95% / 11.875% ≈ 8才是新证据的支持力度。意思是：观察到"高智商"这个证据后，"夜神月是凶手"的可能性相比观察前提升了约8倍。这里为了方便理解，做了简化的表述。
