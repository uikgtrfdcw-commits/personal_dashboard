# 从 AlphaFold 到 ChatGPT，为什么人工智能更多依赖「暴力穷举」而非「优雅建模」？

- 日期: 2025-07-26
- 赞同: 0

人工智能的思维方式和人类一脉相承。在漫长的进化历史中，人类自身发展出了两种截然不同的解决问题的方式，一种是“建模”，一种是“穷举”。
前者的核心是一种优雅的信念：万物背后皆有其理。人类相信，只要足够聪明，他们就能洞察问题的本质，建立一个精准的模型——一个公式、一个蓝图或一个理论。这就像是解一道数学难题，目标是找到那个唯一的、普适的“解析解”。这是一种智力上的一劳永逸，它不仅告诉你“是什么”，更让你明白“为什么”。
这种方法的魅力是无与伦比的。想想牛顿，单凭笔纸和思想实验，就推导出了万物法则。一旦模型被建立，它就拥有了惊人的力量。未来再遇到成千上万个同类问题，只需将变量代入公式，答案便唾手可得。
但这里有个麻烦的地方：建模实在是太难了。构建一个完美的模型，可能需要耗费数年甚至一生的才智。而且模型毕竟只是对现实的简化，一旦忽略了其中某个关键细节，整个宏伟的理论大厦就可能轰然倒塌。
于是，人类发展出第二种应对复杂世界的方式：不去强求看穿本质，而是直接从结果出发，在大量实例中寻找最优的解法。这种思维方式看起来更“笨”，更朴素，但却无比务实——我们可以称之为“穷举法”，或者更准确地说，是“结构性试探”。
它不再依赖先验的规则和推理体系，而是依靠一种更具实践感的策略：试错。但这里的“试”，并不是无方向地瞎碰乱撞，而是在真实世界的隐性结构中，顺着环境早已刻下的沟壑滑行。这就像在满是沟渠的沙地中放一滴水，它看似随意流动，其实总会汇聚到那条最低洼的路径上。
现代科技，尤其是人工智能的崛起，将后一种方法的威力展现得淋漓尽致。比如，AlphaFold 解决了困扰生物学界数十年的蛋白质折叠问题，它没有依赖生物物理学的某种完美模型，而是训练了一个庞大的神经网络，喂入上百万个结构实例，让系统自己“试”出折叠结果。它也许并不“理解”氨基酸链如何变形，但它在亿万个案例中学会了怎么“预测”——这就是实践的力量。
对于那些极其复杂、难以建立模型的“黑箱”问题，这种暴力美学往往是唯一可行的通路。当然，它同样要付出代价，只不过它消耗的不是智力，而是算力资源。随着问题变得复杂，可能性组合的数量会发生“爆炸”，需要尝试的次数会达到天文数字，即便是最强大的计算机也可能力有不逮。而且，这种方法通常“知其然，而不知其所以然”，下次遇到一个新问题，你还得把整个笨重的试错过程再走一遍。
这两种思维方式，象征着人类和这个复杂世界打交道时的两手准备——一手靠猜，一手靠试。一种想明白了再动手，一种边做边看结果。你可以把它们理解为计划与试错、理性与经验、演绎与归纳。在科学、管理、投资、法律等众多领域里，这种对峙和博弈无处不在。
过去几个世纪，科学的进展很大程度上依赖于建模思维。牛顿的力学体系、麦克斯韦的电磁场方程、达尔文的自然选择理论，这些都是试图通过建立模型来捕捉世界的本质。但随着社会复杂度的提高、数据量爆炸、计算能力的飞跃，科学家们开始越来越多地转向“穷举”。从蛋白质折叠到基因编辑，从天气预测到新材料发现，这些高度复杂的问题常常难以构建完美模型。于是人们开始依赖机器学习、统计模拟和自动化实验平台，在大量数据中“找规律”。你可能并不知道为什么这个参数组合有效，但你知道它确实有效。这种从“因果推理”向“模式识别”的转变，是现代科学的一个重要趋势。
在控制工程领域，这种思维的演化同样显著。传统方法强调建模，工程师必须精确理解被控对象的物理过程，用微分方程刻画系统，再推导出控制策略。但对于像等离子体、仿生机器人这类高度复杂的系统，模型往往难以构建。此时，强化学习等数据驱动方法开始崭露头角。AI无需理解底层方程，只需在和被控对象的互动（或者仿真环境）中反复试错，根据反馈调整策略，最终学会“怎么做是对的”。DeepMind控制核聚变等离子体的成果正是如此——AI不解其理，但因试得足够多，最终达到了远超传统建模方法的控制效果。
在管理学领域，曾经的战略模型，如波特五力分析、SWOT工具、BCG矩阵等，都试图帮助管理者从静态结构中“洞察全局”，提前制定详尽的行动计划。管理者们倾向于自上而下的顶层设计，力图以理性规划掌控市场变量。而今天，面对愈发动态和不确定的商业环境，越来越多的管理者更倾向于“走一步看一步”——用最小可行产品试水，用AB测试不断优化决策。他们不再一开始就追求完美计划，而是将管理转化为试验、反馈和迭代的循环。这不是因为他们不懂模型，而是因为他们知道，很多时候模型无法预测现实的所有可能。
在公共治理领域，类似的思维转变也在悄然发生。过去的政策制定，无论是在计划经济体制下的集中部署，还是在战后美欧日的国家发展战略中，政策制定者普遍假设可以通过顶层设计和理性分析，对社会与经济进行精密调控。这背后都反映出了一种“先规划、再执行”的理性主义传统。然而，随着全球经济的不确定性上升、社会系统的非线性增强，这种理性控制的幻觉正在瓦解。如今，面对一个高度复杂、动态、难以预测的复杂系统，政策制定者越来越倾向于“走一步看一步”——以试点取代规划，以迭代取代预测。在现实面前，模型不再是万能剧本，而是可随时修正的工具。
在投资领域，这两种思维的博弈同样激烈。传统的价值投资，正是“建模”思维的典范。投资者试图通过深入分析一家公司的基本面，构建一个关于其未来现金流的精确模型（如DCF估值法），从而计算出其“内在价值”，并在此基础上做出买卖决策。这种方法试图理解商业的本质，并相信价格终将回归价值。而另一端，则是以文艺复兴科技等为代表的量化投资。他们几乎放弃了对“为什么”的探求，转而利用强大的算力，在海量的历史数据中进行“穷举式”的模式挖掘，寻找那些能够带来超额收益的统计套利机会。他们可能不知道某个策略为何有效，只知道它在回测中表现优异。
不过有趣的是，如今，越来越多的投资机构正试图融合两者，用基本面模型进行初步筛选，再用量化策略来优化交易执行和风险管理。
也就是说，在实际工作中，穷举和建模往往并非孤立存在，而是常常结合使用。当纯粹的穷举方案，因可能性太多而无法执行时，我们可以先建立一个简化模型，用它来“剪枝​”，即排除掉大量明显错误的选项，从而大大缩小“穷举”的搜索范围；反过来，当你无法直接推导出完美模型时，可以先通过小范围的“试错”来观察规律，然后基于这些数据拟合出一个近似的初步模型。
从这个角度看，AI 本身或许就是建模与穷举这两种认知方式的融合。穷举能拓展认知范围，用数据和算例探索新的可能；而建模能提高效率，用更少的数据、更少的资源来获得更可靠、更可解释的结果。我们用穷举探索世界，然后用建模提炼规律，最终再用规律来指导新的穷-举。
这也许就是人类处理复杂世界最深层的智慧：我们在混沌中勇敢试错，在试错结果中寻找秩序。在实践中积累经验，又在经验中提炼模型。我们在建模和穷举之间来回穿梭，用双轨并进的方式，一点一点逼近真相。
建模给我们信心，穷举给我们力量。一个教我们如何思考，另一个教我们如何前行。复杂世界没有标准答案，但善用这两种工具，也许可以帮我们走的更远。
注：
或许有人会问，为何提问中要用“暴力穷举”这个略显夸张的词？因为它比“拟合”或“试探”等术语更直观，更能抓住这种方法的精神内核。无论是监督学习式的拟合，还是强化学习式的试探，其本质都是一种试错，一种依赖海量计算来探索所有可能性的高级穷举。这或许才是这场人工智能革命最深刻的注脚：我们并没有发明一种全新的、超越人类的智能形式，而是将人类最原始、最依赖实践的“试错”能力，通过前所未有的计算力量，放大到了一个足以震撼世界的尺度。
