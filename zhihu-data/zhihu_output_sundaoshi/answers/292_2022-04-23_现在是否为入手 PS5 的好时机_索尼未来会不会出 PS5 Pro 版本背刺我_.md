# 现在是否为入手 PS5 的好时机，索尼未来会不会出 PS5 Pro 版本背刺我？

- 日期: 2022-04-23
- 赞同: 0
- 评论: 0

---

如果用百分制来衡量游戏体验，那么ps4～ps5pro就是80～90分的选手。没买就是0分，买了直接上80。
这就像考试成绩一样。及格不及格有很大区别。而80分和90分，区别有，但不大。
再往90分以上，那就不只是游戏机型号的问题了，还有电视机型号的问题… 到时候你就会开始操心，oled和mini-oled屏幕谁更性价比？投影仪和电视选哪个？投影仪要不要上激光？投影仪都买了，再来个菲涅尔幕布吧！更别提音响效果、声场布局、家具摆放… 你干脆换房得了。
设备越好，花钱越多，边际效用就越低。最影响游戏体验的还是游戏本身。就像好听的歌放在糟糕的播放器里也同样好听，游戏不好玩，放在电影院屏幕上也没用。
只要有足够多的时间，遇上足够好玩的游戏，手里又有一台足够好用的设备，那么就足够留下一段美好的回忆啦。
…
其实不只是游戏机，还有像绘图工具啦，篮球运动员的身高啦，子女教育啦，智商水平啊，财务状况啦，都是足够好就可以了。万物皆有边际，何必事事求全。想清楚自己最想要什么，其他的，不如坦然放弃～
还有小伙子你要珍惜光阴啊！等上了岁数，就真的变成买了收藏，只看不玩了… T_T
