# 既然说集体的力量大，那么个人主义是如何走上历史舞台的呢？

- 日期: 2025-07-23
- 赞同: 0
- 评论: 0
- 链接: https://www.zhihu.com/question/1931269675750234005/answer/1931269810106369455

---

尼采说，“成为你自己。”
今天，这句话听起来理所当然，但在过去，它被认为是最激进、最颠覆，也最引人非议的观念之一。在人类历史上的绝大部分时间里，“你”首先不是你自己，而是家族的晚辈、教会的羔羊、国王的臣民。你的身份、你的意义、你该走的路，都是被预先规定好了的。
那么，我们是如何从一个完全服从于集体意志的世界，走进一个可以相对自由地选择伴侣、职业和生活环境的世界呢？更重要的是，在这场天翻地覆的变革中，我们到底得到了什么，又失去了什么？
我们知道，在人类历史的前半段中，人们的道德行为并不来自个人意志，而是由更高的集体力量所规训——比如孔子强调的家族孝道、奥古斯丁确立的宗教秩序、以及马基雅维利所描绘的国家权力体系。这些宏大的叙事，把人的一生牢牢嵌入到社会结构之中，让每个人必须在既定的角色中服从与献身。
所以，从文艺复兴、启蒙运动到科学革命，人类一直在努力“争取自由”，摆脱传统的枷锁——国王的统治、教会的束缚、封建宗族的捆绑。终于，“大写的人”站起来了。人们开始相信，世界的秩序不应再由上帝、国王或祖先的教诲来定义，而应由我们自己，由无数独立、平等的个体来共同探索。
这就是启蒙思想的由来，也是资本主义精神的核心。一个根植于人性、人人为自己负责的价值观。它相信，不论出身贵贱，每个人都能以相对平等的身份参与竞争。当个人的创造力和积极性被源源不断地释放出来时，人类社会也迅速迎来了繁荣。
人类成功了，梦想实现了。今天，城市中生活的人们，远比祖先更加独立，更有选择权。但这份来之不易的自由，却也带来了一个全新的、沉重的心理负担。当那些旧的、集体性的“保护壳”都消失后，每个独立思考的个体，都必须独自面对一个庞大、冷漠、充满不确定性的世界。这意味着前所未有的孤独、无力和焦虑感。
为了摆脱这种难以忍受的焦虑，现代人开始不自觉地“逃避自由”——正如心理学家弗洛姆所观察到的，为了消除与世界格格不入所带来的恐惧，人们会主动放弃自己真实的、自发的个性和感受，转而像一台机器一样，全盘采纳社会文化为他准备好的“标准配置”。他变成了周围环境的倒影，通过让自己变得和别人毫无差别，来消解孤立无援的痛苦，重新获得融入集体的安全感。
当你理解了弗洛姆的这个观点后，历史上的许多思想家，似乎一下子都串联了起来。弗洛姆就像一个“嘴替”，精准地表达出了大家都在思考，但未曾系统化表述的东西。比如克尔凯郭尔的焦虑，萨特的自欺，昆德拉的媚俗，以及爱默生所批判的盲从，它们都是现代人在“自由的重负”下，所呈现出的不同症状。
某种角度上讲，这些人都是个人主义的信徒。
一个社会的主流价值观一旦向个人主义靠拢，就会对其中的每个人提出更高的要求：你要独立思考，追求你的事业，并为自己的选择负责。它鼓励你做命运的主人，而不是上帝的仆从、家族代言人，或者社会阶梯上不停攀爬的螺丝钉。
然而，世界上并不存在完美的终极答案，个人主义也有其引人争议的另一面。
首先，世界变得无限多样，也因此充满了不确定性。每个人都要走自己的道路，最能创新的人往往也是最“疯狂”的人。但很多时候，疯狂就仅仅是疯狂，离经叛道者未必有真知灼见。在这样一个更有活力、更加多样化的社会里，每个人都要学会容忍自己看不惯的人和事。
其次，自由竞争带来了巨大的贫富不均。我们必须承认，人和人之间的才能与品格千差万别。一定程度的贫富分化，是市场激励机制起作用的表现。但是，过大的贫富差距会撕裂社会共识，导致社会不稳定。
最后，个人主义意味着每个人都要为自己的行为承担全部责任。这不仅要求普通人，也要求精英阶层不能再无限免责，无法再轻易地隐身于权力或集体的神圣光环之下。
这就是个人主义带给我们的礼物，既令人兴奋，又无比沉重——无与伦比的创造力、前所未有的个人自由，以及亲手塑造自己命运的权利。而这同时也意味着前所未有的挑战——背负自身命运的巨大压力、与人群格格不入的深刻孤独，以及一个赢家和输家并存的社会所带来的尖锐冲突。
长江之水不能倒流，集体主义的宏大叙事已经渐行渐远。某种意义上讲，我们这一代人面临的真正难题，除了如何摆脱枷锁，更是如何拥抱自由。
