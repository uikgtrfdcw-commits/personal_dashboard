# 「中国税负远超欧美」的说法是否属实？

- 日期: 2025-12-23
- 赞同: 0
- 评论: 0

这话对也不对。
根据世界银行数据，法国、丹麦的宏观税负（官方统计税收+社保缴费）的GDP占比常年超过46%，瑞典接近44%，即便以"低税"著称的美国也高达27%。
中国的名义宏观税负大约在18%到20%之间，远低于西方国家。但这个数字只统计了纳入官方口径的名义税收（增值税、企业所得税、个人所得税等）和社保缴费。还有一些政府收入并不体现在这个统计里，却同样从居民口袋里转移走了财富——我们可以称之为"隐性税费"。注1
首先是土地财政。地方政府通过出让土地获取收入，而土地成本最终体现在房价里。当你花五百万买一套房，其中可能有两百万以上流向了政府，只不过它不叫"税"，叫"土地出让金"。这笔钱不计入宏观税负统计，却实实在在从你的口袋里转移走了。
第二是垄断行业的超额利润。电信、石油、金融、烟草等行业的国企凭借垄断地位获取高额利润，消费者为此支付的溢价本质上也是一种隐性税。你每次加油、打电话、办业务，都在向这个系统缴税。注2
第三是通货膨胀。政府通过增发货币稀释债务，而货币贬值的代价由所有持有现金和存款的人承担。这是一种无需立法、无需通知、人人都无法逃避的隐性赋税。注3
第四是各种行政收费和合规成本。企业为了获得许可证、通过审批、应对检查而支付的费用，最终都会转嫁到商品和服务价格中，由消费者买单。注4
如果把这些隐性税收都算进去，中国的实际税负并不比西方低多少，有些学者估算甚至接近35%到40%。差别在于：西方的税痛在明处，你知道自己交了多少、交给了谁、用在了哪里，可以监督问责；而隐性税负痛在暗处，你感受到生活成本高、存款贬值、房价离谱，却很难指出具体是谁拿走了你的钱，更不知道钱花在哪里。
除了隐性负担，不合理的税制结构也会塑造政府的行为逻辑。中国的第一大税种是增值税（占比约30%），根据企业销售额征收；而与居民收入直接挂钩的个人所得税仅占很小比例。这意味着政府的财政收入更多取决于企业规模和交易总量，而非居民实际拿到手的收入。
这种税制结构看似利好民众，但实际上抑制了居民收入。因为政府的税收大部分来源于企业销售额，因此政府倾向于做大企业、推高GDP，而不是提高居民的可支配收入。反观西方国家，个人所得税往往是第一大税种，政府要想多收税，就必须让民众挣得更多。注5
从宏观数据也能看出端倪。目前中国居民消费占GDP的比重只有38%，不仅远低于美国的68%、英国的64%，更低于全球平均的55%。这意味着在整个经济产出中，真正流入居民口袋、由居民自主支配的部分异常之少。大部分的钱被政府和企业（尤其是国有企业）拿走。
正如阿西莫格鲁所说，问题的关键不在于政府拿走多少，而在于拿走的方式是否合规透明、是否回流为公共服务、是否接受监督。一个税率50%但规则清晰的制度，可能比一个税率20%却处处吃拿卡要的制度更加包容，更加有利于经济的长远发展。
需要注意的是，随着近几年大规模减税降费（如增值税留抵退税）和房地产市场下行（土地出让金大幅缩水），大口径宏观税负有所回落，从2018年的37%左右下降到了2022-2023年的32%左右。
西方国家
话虽如此，其实高税率本身未必是坏事。西方发达国家普遍高税率。
前面说过，根据世界银行数据，法国、丹麦的宏观税负（官方统计税收+社保缴费）的GDP占比常年超过46%，瑞典接近44%，即便以"低税"著称的美国也高达27%。
西方国家为何要征收如此高昂的税？既然阿西莫格鲁在《国家为什么会失败中》说"汲取性制度"不好，那为什么发达国家反而税率更高？
答案是"汲取性制度"并不等于"高税率"。
衡量一个制度的好坏标准，不在于税率的高低，而在于产权的保护程度。换句话说，真正扼杀经济活力的不是政府拿走了多少钱，而在于拿走的方式是否遵循可预期的规则，以及这笔钱最终流向了何处。
举个例子。1688年光荣革命之后，英国的宏观税负在欧洲首屈一指——1700年前后约为9%到10%，到拿破仑战争时期攀升至18%到20%，同期法国约为12%，普鲁士不到8%。然而，恰恰是这个"高税收"国家率先完成了工业革命，经济领跑全球。原因在于，英国最早确立了"无代表不纳税"的原则，商人和地主在议会中有自己的代表，自己同意给自己加税。而且税款中相当一部分被用于建设海军以保护贸易路线，建立法庭以执行商业契约。对商人而言，这是一笔昂贵的"商业俱乐部会员费"，虽然交得多，但确实能换来资产安全和稳定的商业环境。
反观同时期的中国，明清两朝虽然定下了极低的名义税率——明代官方田赋约为产出的3%到5%，清代"永不加赋"政策后低至2%到4%。但这只是正税，实际上，农民还要承担火耗、摊派、徭役等名目繁多的隐藏赋税，真实税率远高于账面数字。而且商人的处境更为凶险。沈万三富可敌国，却被朱元璋抄家发配；胡雪岩纵横商场数十年，最终被革职追债、变卖家产，贫病而终。连首富都可以因皇帝一句话而倾家荡产，普通人的财产更是朝不保夕，长远的经营策略根本毫无意义。这种不可预期性，才是阿西莫格鲁所说的"汲取型制度"。
此外，西方国家的税率并非天生就高，而是经历了一个逐渐演化的过程。
19世纪末之前，多数欧洲国家的宏观税负只有5%到10%。英国是个例外——拿破仑战争期间为筹措军费开征所得税，税负一度攀升至近20%，但战争一结束就废除了，因为民众认为政府没有理由在和平时期拿走这么多钱。到19世纪中后期，英国的税负也回落至10%左右。那时的政府职能极其有限，主要就是维持军队、警察和法院，偶尔修修路。
第一次税率提高发生在两次世界大战期间。战争需要全民动员，政府开支急剧膨胀，税率随之飙升。到1945年，主要参战国的宏观税负已经从战前的不到15%翻倍至25%到30%。战争结束后，这些税率并没有降回去——战后欧洲各国普遍爆发了民权运动，导致政府不得不提高福利待遇支出。
第二次税率提高发生在战后到1980年代。在与苏联模式竞争的压力下，西方国家纷纷建设福利国家，税收的去向从军备转向了医疗、养老、教育和失业救济。到1970年代末，瑞典的宏观税负已经突破50%，法国、德国也逼近40%。
1980年代里根和撒切尔的减税改革短暂扭转了这一趋势，但刚性的福利支出使得税负很难大幅下降。此后几十年，西方国家的宏观税负基本稳定在35%到47%之间。
换句话说，今天欧洲的高税率是过去一百多年累积的结果。每一次战争、每一项福利承诺，都把税率往上推了一个台阶，而降下来却极其困难。
注：
1. 隐性税负
文中“隐性税负”的说法是为了方便理解，在学术上并不严谨。
严格来说，土地出让金、国企垄断利润、通货膨胀效应以及行政合规成本，均不构成税法意义上的赋税。
其中，土地出让金以及国企垄断利润中上缴政府的部分，可被视为“准财政汲取”或“广义财政收入”；而通货膨胀和行政合规成本并无明确的财政流向，更接近于居民部门在既定制度环境下承担的系统性经济损耗，政府从中间接获益。
本文“隐性税负”的简化表述，旨在强调这些机制在经济效果上与税收具有相似性——即通过制度安排，将居民部门的收入或福利系统性地转移至政府或国有部门，而非对其进行严格的税收概念界定。
2. 国企
美国国有企业被称为联邦公司，仅占GDP的1%，主要集中在航天、邮政、公共设施等市场激励不足的领域，且管理严格，不能上市，也没有竞争特权。
相比之下，中国国有企业GDP占比约为25%，在土地、能源、资金等要素的获取上，国企通常享有优先配置权。在烟草、石油、电信、电力、铁路、航空、金融等关键行业长期垄断，民企基本难以进入。同时，国企还经常享受财政补贴、税收减免和监管豁免。即使经营效率低、债务高企，也能获得政府“兜底”保障。
此外，在金融资源配置维度，国有金融机构控制着超过三分之二的金融资产和信贷投向。
2. 通货膨胀
美国同样面临通货膨胀问题，但在制度安排上，通胀上行通常伴随着名义工资的阶段性调整，社会福利项目（如养老金、失业救济）与通胀指数挂钩，税制中的税档和扣除额也会随价格水平变化进行自动修正，从而在一定程度上缓冲通胀对居民实际购买力的冲击。
相比之下，中国相关调整机制的覆盖范围相对有限，且居民资产结构中现金储蓄和房地产占比较高，在通胀环境下，居民部门的实际购买力更容易受到直接侵蚀。
3. 行政成本
美国同样存在行政合规成本，但相关流程通常高度标准化，规则边界和责任划分较为明确，大量制度摩擦通过财政供养的公共部门和公共服务体系在内部消化。
在中国，一些基层行政事务在执行速度上可能更高，但规则解释空间较大、流程调整较为频繁，企业和居民往往需要自行投入额外时间、人力进行沟通、准备和协调。
4. 增值税
在现行政治体制下，以企业和交易为主要征税对象，比以居民收入为核心的直接税制更易实施，居民痛感更不明显，阻力更小。
