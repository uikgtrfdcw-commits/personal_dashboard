# 中国为什么选择了渐进式改革道路?

- 赞同: 0

“渐进式改革”不是凭空发明的，而是有着深刻的历史教训。法国大革命就是最典型的反面案例。
首先，为什么一定要改革？沿袭祖宗之法，老成谋国不好吗？
答案是到了1780年代末，波旁王室已经无法掩盖多年来积累的深层结构性问题。无论路易十六多么勤政爱民，财政危机依然愈演愈烈。旧制度本身的结构已经与时代发生了错位。就像一台只适用于简单任务的旧电脑，被迫运行一个复杂多元的现代系统，最终的结局只能是崩溃。
波旁王朝的政治精英们，认知上早已和时代脱节。他们成长于一个封闭、稳定、自我感觉良好的环境，他们的世界观和政治经验都来自那个早已消逝的时代。他们很难真正理解新兴阶层的诉求，常常把人们要求权利的声音误认为是颠覆政权的阴谋。他们既不具备与新社会沟通的语言，也缺乏平等协商的思维方式。
有人对此讽刺道（据说是托克维尔，但其实是误传）：
...他们的知识结构和价值观念永远停留在青少年时期的某一阶段。他们用古代的理念去解决今天的问题，并固执的认为自己捍卫了某种价值。他们穿着古代的戏装在现代舞台上表演，像光天化日之下的幽灵。人人都知道他是幽灵，他们却自以为是真神。
其次，波旁王朝在制度上也早已僵化。那些延续下来的等级制、官僚体系，以及对社会流动、言论与司法的限制，本质上不是用来满足社会需求，而是为了压抑和控制民意。这种制度像一块坚硬的玻璃，面对越来越大的外界压力时，既无法吸收，也无法缓冲，裂纹越来越大，最终碎裂已经是迟早的问题。
在操作层面上，波旁王朝更是捉襟见肘。习惯了颁布命令的财政官僚和宫廷贵族们，往往对辩论、妥协、共识一无所知。他们不懂得如何和平处理反对派的挑战。他们习惯了用命令、收买和恐怖解决问题，不懂得如何通过协商构建秩序。
站在今天，我们不得不思考：如果旧制度真的已经无法承载社会发展的能量，那改革的真正目的该是什么？如果我们穿越回去，应当如何应对当年法国的窘境？
答案并不是简单地“避免爆炸”，而是建立一个更稳定、更有弹性的现代秩序。真正的改革，不只是换一个人当权，不是把旧制度涂一层新漆，而是要从根本上解决三个问题：合法性、冲突化解机制、以及系统的自我调整能力。
合法性的核心，是权力来源于人民的同意，而不是源于暴力或家族传承。只有当社会广泛认可政权的正当性，这个政权才能真正的稳定。反过来说，如果一个体制只能靠枪杆子来维持，那就注定会有下一个枪杆子更粗的人来轮流坐庄。
其次，现代社会的矛盾和分歧是常态，指望消灭冲突是不现实的。改革的目的，是为这些冲突提供非暴力的表达渠道，让“你死我活”的敌对关系，变成“有规则的比赛”。能否构建一个和平解决分歧的制度安排，决定了一个政权能不能在现代、多元、复杂的社会中持续运转。
最后，任何系统都会犯错。当愤怒积累到极限，如果没有一个安全释放压力的机制，最终的结局就是一次剧烈的爆炸。因此，一个健康的体制，必须拥有自我纠错的能力，哪怕走错了路，也能调头回归正轨。
旧制度像一个密封的高压锅，它依靠厚重的外壳来压制不断升温的内部压力。但再厚的外壳，也终有一天会被冲破。相比之下，成功的现代改革，更像一个工业反应堆，装有冷却系统和阀门，能主动调节和引导内部能量，从而避免爆炸，实现持续产出。
不幸的是，在法国大革命前夕，由于多年来积累和压抑的愤怒和戾气，路易十六的一些看似“开明”的改革举动，反而成了掀起革命洪流的导火索。事实上，与前辈比起来，路易十六远非最残暴的君主。他在位时，法国的政策比以前开明，经济也在复苏。然而，革命偏偏就在这个看起来相对“宽松”的时刻爆发了。这也是威权统治的诅咒——当你习惯了用暴力和恐怖来维护秩序，一点点的缝隙都会带来巨大的不稳定性，于是你只好采用更激烈的手段。
结果是，在愈演愈烈的财政危机下，路易十六被迫召开停摆已久的三级会议，结果却变成了第三等级觉醒的契机。他们第一次在合法框架下组织起来，发现自己不仅人数众多，而且拥有清晰的利益诉求。一场原本只是为了“引流”的改革，结果炸开了整个水坝，首先被吞没的正是那个想修补水坝的国王。
但问题随之而来：如果旧制度的匆忙改革往往导致更大混乱，那制度以内的人就只能坐以待毙吗？是否真的存在那些既没有激烈革命、又实现平稳过渡的正面案例？
答案是肯定的。
19世纪的英国议会改革，堪称是“旧制度自我更新”的典范。十九世纪初，工业革命如火如荼，催生出一批新兴中产阶级和工人阶级，对当时由贵族和资本寡头控制的议会施加了巨大压力，英国面临着类似法国大革命的风险。在目睹社会巨变的现状后，英国统治阶层并没有像法国王室那样死守旧制，而是由一部分贵族出身的开明派，特别是辉格党人，主动开启了长达数十年的渐进式改革。他们没有等到社会彻底失控才出手，而是在政权还拥有权威、议会制度尚且稳定之时，就看清了不改革的危险。这种主动性和前瞻性，是改革得以顺利启动的先决条件。
更重要的是，英国改革并不是毫无章法地开放政治空间，而是在已有的法治框架内，对选举权与议会结构进行一轮轮小心翼翼的调整。1832年、1867年、1884年的三次改革法案，每次都只扩大一部分人的投票权，先是中产，再是工人，最后是农民。这种制度内的渐进修补，不仅降低了不确定性，也让整个社会逐步适应了政治参与的过程。
在这个过程中，辉格党并非单打独斗。他们巧妙的拉拢了社会各阶层，配合改革。一方面安抚保守派，保证改革不会“一步到位”地冲击贵族利益；另一方面吸纳体制外的新兴力量，尤其是商界和城市知识分子，让他们在新的政治结构中拥有切实的利益与话语权。与其让这些人去“另起炉灶”，不如让他们“共同参与”。
正因如此，连马克思后来都承认，在英国这种有议会传统的国家，工人阶级完全可能通过和平的议会道路获得权力，而无需诉诸法国式的暴力革命。比如马克思 1872 年在荷兰海牙国际工人协会大会上说：
 “我们知道必须考虑各国的制度、习俗和传统，工人阶级在取得胜利的方式上并不是到处都一样。在英、美和荷兰这样的国家，工人们或许可以用和平手段达到目的。”
而西班牙在1970年代的转型，则更是一场教科书般的、体制内主导的和平改革。当佛朗哥独裁者去世后，他所留下的旧体制，并没有硬撑到底，也没有在恐慌中解体，而是由接班人卡洛斯国王和首相苏亚雷斯主动出手，拆掉了旧制度的顶层结构。他们并没有彻底打烂旧体制，而是利用佛朗哥时代的法律工具，通过一部《政治改革法》让独裁议会投票，自己废除自己，从而实现了“在法律中完成革命”。
这背后体现出的不仅是技巧，更是一种极强的政治敏感。他们深知，如果没有稳定的制度路径来承接社会发展带来的活力，社会很可能会走向动荡和混乱。所以他们一方面设法保障了反对派的合法性，邀请共产党、社会党坐上谈判桌；另一方面，也安抚了军队和右翼旧势力，承诺不清算旧账、不搞血腥清算。在改革正式启动前，他们等于先修好了河道，再一点点放水。
回过头看这两个案例，不难发现它们背后的共同逻辑：改革不只是一场道德冲动，更是一套严密设计的制度工程。它需要主动预判、制度先行、妥协合作、渐进推动。那些主动发起改革的体制内精英，并不是良心发现，放弃权力，而是明白“抓得越紧，自己越危险”的道理。与其在一夜之间被暴力夺权，不如提前、可控地释放部分权力，把变革的洪水导入可以承受压力的新河床。
改革不可能没有代价。但只要执政者拥有政治远见，善于构建规则，懂得如何与对手谈判、与盟友协商、与现实妥协，那改革不仅可能避免灾难，还能让一个垂暮的旧体制，在阵痛中获得新生。
那么问题来了，改革只能寄希望于体制内精英的高瞻远瞩吗?
答案并非如此。如果历史告诉了我们什么道理，那就是寄希望于“明君开恩”是不现实的。所谓“观念的水位决定制度的变迁”，只有当公民社会逐渐成长，民众通过维权、社区、公益和网络议政等方式学会协商、合作、运用法律、表达诉求，从“臣民”变为“公民”，社会才有变革的可能。一个成熟的公民社会，不仅能在日常公共事务中形成持续的监督和改革的压力，也能在顶层松动或突变时，迅速填补真空，避免社会结构陷入无序和混乱。
理想的情况下，成功的和平转型，往往是一场“钳形攻势”：底层的公民社会是发动机，提供动力和压力；顶层的开明精英是转向阀，提供路径和渠道，二者缺一不可。没有底层的压力，改革很容易停滞甚至倒退，最终沦为“表面改革、实则维稳”；而没有顶层的配合，底层的抗争则很可能演变为失控和混乱，重回军事强权的循环。
这也是为什么，某些国家在改革时，并没有选择“打碎重来”的休克疗法，而是通过特区试点、分步推进、在旧制度框架内逐步释放社会活力，把社会矛盾引导进可控的制度轨道。
只不过，历史不会因为一次选择就一劳永逸。渐进一旦失速，社会的期待和制度回应之间就会出现危险的真空。当改革步伐放缓甚至倒退时，问题并没有消失，只会压抑和累积。高压锅不会因为沉默而降温，洪水也不会因为围堵而退去。法国的未来走向何方，我们仍然拭目以待。
