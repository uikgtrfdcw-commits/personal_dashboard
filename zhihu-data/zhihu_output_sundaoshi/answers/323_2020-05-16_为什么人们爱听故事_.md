# 为什么人们爱听故事？

- 日期: 2020-05-16
- 赞同: 0
- 评论: 0
- 链接: https://www.zhihu.com/question/19892588/answer/1227128329

---

因为故事是一种催眠。
换句话说，故事能以一种含蓄的、间接的、启发性的方式，给人以心理暗示而造成影响。
我们都有这个经历，本来聊天聊的好好的，一旦话题到了自己身上，心中的防御机制就会咯噔一下启动。也正是因为这个原因，我们很难直接说服一个人。
而听故事的时候，由于故事里的主人公不是自己，故事就会很容易绕过读者的心理防线，轻易实现共情。故事会潜移默化的影响我们的潜意识，而潜意识决定了我们的行为模式。
一个好故事会吸引你，影响你，把你带到一个不同的世界。一旦你和故事中的人物产生共鸣，接受其心理暗示，你的人生脚本就会发生改变。
我高三有段时间特别容易紧张，一到模拟考就手心发热，额头冒汗，胡思乱想，根本停不下来。最后问题是怎么解决的呢，说起来也比较好笑：
有次考试走神的时候，我头脑中忽然浮现出了动漫里的旗木卡卡西，想起了他在面对压力的时候，如何从容应对，分析局势，化解危机。然后我想起了他那个淡定的眼神…
然后我就淡定了
后来我考试基本上就是这个表情...
听故事的时候，我们会不自觉的把自己代入到故事角色的位置，见证他的喜怒哀乐，为人处事，以及角色本身的成长。一个好的故事角色，会成为我们心灵的老师。
故事就像一面镜子，反映出我们内心中最为隐秘而本质的角落。那些你最难以忘怀的故事情节，仔细分析的话，往往都对应着你内心中的某种强烈而未曾解决的情感讯息。
所以，如果善于观察和反省，一个好的故事往往能够启发我们，引导我们，让我们主动思考，找到解决问题的信念和勇气。一个好故事带来的影响，将伴随我们的一生。

另外一个角度，现代社会年轻人的生活状态。
城市化运动打破了传统的家庭观念和宗族网络，传统熟人社会中人和人之间的强链接已经不复存在。年轻人在城市中独自生活，远离家庭和亲族的指引，也失去了光耀门楣的意义。加上基督教和左派思潮的式微，年轻一代空前的世俗化，生活在价值观的真空地带。
现代社会能够提供的，是花样翻新的物质满足和感官刺激。互联网公司里最聪明的人每天都在竭力研究如何诱导年轻人消费再消费。
这种大环境下，动漫和游戏的流行并不是一种偶然。心灵的发展需要榜样，需要导师，需要试验的机会。那些好的游戏和动漫，抚慰了青春期的创伤，通过角色扮演和移情使治愈成为可能，帮助我们在一个竞争激烈的、功利化的社会中，交到因为我们是我们而和我们在一起的真心朋友。
