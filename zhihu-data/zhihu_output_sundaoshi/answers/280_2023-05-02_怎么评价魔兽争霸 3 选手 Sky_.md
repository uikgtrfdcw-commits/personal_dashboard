# 怎么评价魔兽争霸 3 选手 Sky？

- 日期: 2023-05-02
- 赞同: 0
- 评论: 0

盖哥并不是最有天分的选手，也谈不上有什么专业的训练条件，最早都是业余时间在网吧混着，没有拿到任何名次。02年wcg西安预选赛，盖哥第一轮就被淘汰，差点从三楼跳下去。
后来ehome的队长逍遥，也是sky在医学院的同学兼好友。他回忆说，sky一次次挑战那些“高不可攀的对手”，然后“整晚整晚的失败”。逍遥亲眼见证，sky一个又一个的战胜这些对手，反而成为人家眼中的高山。他当时不能理解sky为什么能在一次次的失败中乐此不疲，但他已经意识到，这只打不死的小强有着难以估量的潜力。
Sky真的爱这一行，这是他的超能力。和Sky相比，我们很多名义上的热爱，其实更多是对舆论压力的恐惧，或对现实生活的逃避。
真正的热爱，是那件遭受挫折后，你仍能继续投入努力的事情。在这件事上，没人能打败你，因为一遇到挫折，别人就放弃了。
Sky就像卡卡罗特，哪怕面对比自己段位高很多的对手，他也从不畏惧。他屡败屡战，愈挫愈勇，甚至越败越兴奋。这是真正的高手心态。我观察过，自己打魔兽时，连续被挫败的忍耐上限是7次，而解数学题时是3次。超过这个上限，心态就崩了。所以我永远也打不了职业，也拿不了奥数名次，因为不够感兴趣。
在魔兽争霸的世界中，我的战斗力只有7，而sky至少有一万以上。
不过我想，每个人都有属于自己的魔兽世界。每个人都有自己的呼召、激情和热爱。只要耐心尝试，持续探寻内在最深邃的冲动，你总能发现蛛丝马迹，开启新世界的大门。叩门的，就给开门，寻找的，就必寻见。
然而不得不承认，sky的成功太偶然了，偶然到任何一个变量稍加偏转，他就可能跌入万劫不复的深渊。从许立志到聂兆威，现实中这样的例子不要太多。天堂与地狱通常只有一步之遥，运气也是实力的一部分。我们可以想象，一定有成千上万个更努力、更有天分的sky，只是因为运气稍差，便一生默默无闻。或许，我们不应该用单一的价值体系去衡量所有人。在自己热爱的事上拼搏过，不管是否有能力走到最后，都已经是赢了。
Sky的成功不只是一个底层小孩的逆袭，更是一代年轻人对社会偏见、对命运的抗争。在这场抗争中，没有人是失败者。
Sky、林书豪、年初的Oliveira、还有前几天来京签售的西塞罗，他们在一个不属于自己的时代，摸索出了一条属于自己的道路。他们的故事会成为传奇，激励一代又一代的年轻人：
要相信自己所热爱的事物，坚持做下去，它便带你到那命中注定的地方。
