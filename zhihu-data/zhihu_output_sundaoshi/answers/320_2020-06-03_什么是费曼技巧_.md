# 什么是费曼技巧？

- 日期: 2020-06-03
- 赞同: 0
- 评论: 0
- 链接: https://www.zhihu.com/question/20585936/answer/1262942925

如果你不能用简单的语言来描述你学到的知识，那么你就并没有真正理解。
只有用最简单的语言，才能把所有的
底层逻辑链条展示清楚
。如果你不得不使用一些行业术语，那么这个术语中包含的细节信息你并不明白。
同理，死记硬背并不是理解。如果一个家伙老是把专业术语挂在嘴边，可能他对自己的底层逻辑并不是很有信心。费曼本人能用薄薄的一本讲义，从算数、代数、三角函数，一直推导到微积分，整个逻辑链条完整流畅、通俗易懂。钱学森回忆自己在北师大附中的求学经历说，自己临考试是不做准备的，大家都重在理解不在记忆。如果通过死记硬背、急功近利获得高分，同学们反倒瞧不起。
当然了，人们常常会故意把一件事情讲的高深莫测，云山雾罩，这样就显得自己的水平高一些。最好加上数学公式和专业术语，以壮声势。
这就不得不提到周孝正老师的那句名言：
一流学者是自己明白，而且能给外行讲清楚；二流学者是自己明白，而且能给同行讲清楚；三流的学者是自己明白，但不会讲；不入流的学者是自己都搞不明白，就敢出来胡说八道。
让我真正认识到费曼技巧威力的，是每天中午和郭师兄聊天的悲惨经历。这家伙懂的实在太多了，我根本说不过他。为了保证跟得上节奏，我只能前一天晚上背答案。就算这样我也会卡壳，而且我不能保证第二天背下来的东西第二周仍然记得。 
更要命的是，我太喜欢记笔记了。这当然并不全是坏事，我可以很轻松的找到自己两年记下的关于某个话题的鬼畜知识，只要搜索关键字就可以了。但问题是这些知识的逻辑并不完整，我只是知道，但不能表达。记笔记会给你一种自己懂很多的错觉。 
所以记笔记并不能解决全部问题，背笔记更是愚蠢到家。当你发现自己在刻意背诵，那么你并不真正理解。真正有效的做法是表达，用自己的话说出来，把知识内化成自洽的体系。 
以弗所书说，当你说话的时候，要让听见的人得益处。我一向以此为座右铭，没有用的话就不说。但现在我的看法有所改变。如果你想有所进步，那就必须做一个话唠。
长期记忆
费曼技巧是一个
提取短期记忆，转化成长期记忆
的过程。前者是被动的，后者是主动的。
短期记忆是一知半解的知识碎片，看起来脸熟。长期记忆是内化成自洽的逻辑体系，可以脱口而出。一个是硬盘，一个是内存。
学英语的时候，认识的单词很多，但写作能用上的单词却很少。这是因为能够主动使用的词汇属于长期记忆，而长期记忆的容量是相当有限的。
所以，当我们用自己的话来表达一个新学到的概念时，在旁人看来，可能不过是相同概念的同义词替换，但事实上，对自己而言，原本那些模棱两可的
被动词汇
，已经被翻译成了清晰可辨的
主动词汇。
而主动词汇，才是真正属于自己的东西。
