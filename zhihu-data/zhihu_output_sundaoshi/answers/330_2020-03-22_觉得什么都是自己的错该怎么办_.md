# 觉得什么都是自己的错该怎么办？

- 日期: 2020-03-22
- 赞同: 0
- 评论: 0
- 链接: https://www.zhihu.com/question/282370625/answer/1096546566

---

其实如果你仔细回忆的话，自责的感觉并不是第一次出现，而是要追溯到很久很久以前。
刚刚出生时，我们认为自己是世界的中心，世界为我存在，我无所不能。只要我一动念头，世界（母亲）就会立刻给我回应。科胡特给这种心态起了个很形象的名字，叫做『全能自恋』。
但如果世界没有给予回应，我们往往以为是自己搞砸了。当我们看到父亲的暴跳如雷，或者母亲的恐惧和悲伤，就会自发的把一切原因都归结到自己身上。
这就是我们最初的罪恶感，我们的原罪——无法取悦自己的父母。
通常情况下，这种自责倾向会随着父爱母爱的滋养而淡化。当我们从父母那里获得足够的安全感，就能够放下自己，放心大胆地去探索这个世界。我们也会逐渐开始认识到，原来别人是和自己平等的存在，开始和别人建立更具深度的、相互尊重的关系。
但如果我们生活在一个情感贫乏的环境，常常被忽视，常常感到自责，常常体验父母的负面情绪，那就很容易习惯这种思维模式，将心理创伤内化。这样即使长大成年以后，每当遇到生活中的不愉快，也会暂时退化到全能自恋的状态，被自责的情绪淹没。
所以这并不是你的错。你当然会“犯错”，你又不是机器。但如果你的内心充满了爱和力量，你就不会怀疑自己，而是会理性归因，专注于解决问题本身。你什么时候见过科比因为投丢了球而不敢投的？
原谅自己。接纳自己的不完美
。
最后贴一段我很喜欢的话，出自罗素《论幸福人生》：
幸福的一个重要条件，在于随着年龄的增长，渐渐减少了对自己的过分关注，不再为自己的遭遇而感到过分痛苦。对外界事物的关注也会带来痛苦，比如战争、失去朋友等。但对自我的厌恶带来的痛苦往往是灭顶之灾。
一个人有广泛的兴趣，关心整个世界，就会像一滴水归入大海，慢慢忘记了自己的存在，也不再恐惧死亡。
