# 「缺省」这个词是如何从英语 "Default" 翻译过来的？

- 日期: 2020-12-28
- 赞同: 0
- 评论: 0
- 链接: https://www.zhihu.com/question/20953160/answer/1650371479

---

Fault本意是过失、错误。十四世纪的威尼斯犹太人建立银行系统，用fault这个词来指代坏账。而default就是
取消坏账
，也就是不履约。[1]
而在早期编程语言里，fault的意思是
参数缺失
，而default，也就是参数缺失后，自动采取默认参数。[2]
把default翻译成『缺省』这个词，显然是取后一个意思，也就是确失或省略。不知有意无意，这个翻译虽然没有『默认』或者『预设』那么简单易懂，但在计算机领域，倒也算贴合原意。
有趣的是，我看到有人把缺省这个词拿来当自我介绍：『希望自己时常处于缺省状态』。我猜如果按前面一个意思，这句话是指希望自己可以取消坏账，勤于反省。如果按后一个意思，就是不设置外部参数，不用刻意的教条勉强自己。
真是妙啊。
