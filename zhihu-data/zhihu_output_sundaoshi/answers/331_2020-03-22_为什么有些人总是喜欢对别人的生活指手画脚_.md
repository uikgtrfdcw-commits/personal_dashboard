# 为什么有些人总是喜欢对别人的生活指手画脚？

- 日期: 2020-03-22
- 赞同: 0
- 评论: 0
- 链接: https://www.zhihu.com/question/39695205/answer/1096439999

---

愤怒就是无能，指责就是恐惧，对别人指手划脚，就是对自己的不自信。
这就像游泳。把水踩下去，自己就能浮起来。指责别人，就能让自己感觉好过一点。
类似的，在任何社会等级制度中，那些对自己没自信的人，会通过虐待他们眼中的下等人来凸显自己的身份。在美国，底层白人是对待黑人最残酷的群体。
一个人读书越多，阅历越广，对自己和世界的理解越深入，对待别人也就越宽容。罗素说，如果你一听到一种与你相左的意见就发怒，你就要小心了。因为一经检查，你大概会发现，你的信念并没有充分证据，你已经下意识地感觉到，你那种看法没有充分的理由。如果某个人硬要说一加一不等于二，或者说冰岛位于赤道，你就只会感到怜悯而不是愤怒，除非你自己对数学和地理也是这样无知，以至于他的看法竟然动摇了你的见解。迫害常见于神学领域而不常见于数学领域，是因为数学问题是知识问题，而神学问题则仅是立场问题。
一个真正自信的人，对自己和世界的理解足够深入，知道自己是谁，或者渴望变成谁，他就已经拥有了一个成熟稳定的心理内核，建立了独立自洽的自我评价体系。到那个时候，他也就不再依赖于贬低别人来获取廉价的自我认同，看天下无一不是好人了。
…
反过来，指手划脚并不能真的改变别人。
一个人的底层信念，定义了他一切的行为模式。而这种底层信念，只能由人的内心主动得出来。 
比如一个孩子在自私冷漠的环境中长大，他会主动得出结论：“我要不择手段的满足自己的欲望！”表现在行为上，就是暴力和欺骗。这个孩子的行为可以暂时被约束，但其内心的信念不可能因为说教而改变。 
有经验的引领者，这时候不会轻易打骂，而是亲自示范一个宽容有爱的生活。孩子耳濡目染久了，自然会自己得出结论，“似乎对别人好一点也不错。” 
你瞧，一个好的引领者并不会简单的告诉别人应该怎样做，告诉了也没有用。相反，他会巧妙的展示事情的来龙去脉，给人以独立思考的空间。
毕竟，要真正说服一个人，就得让他自己得出结论。
