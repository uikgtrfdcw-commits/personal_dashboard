# 学习是看书还是看教学视频更有效率？

- 日期: 2023-01-06
- 赞同: 0
- 评论: 0
- 链接: https://www.zhihu.com/question/53078038/answer/2832150664

---

视频适合入门，而书籍适合深入学习。
原因很多，比如：
1. 视频比文字更直观。大脑要理解文字，必须先把文字转化成“意念”，也就是大脑中的图像（
翻译专业
的同学对此再熟悉不过）。而视频就可以直接理解。
其实对于专家而言，读书是一种近似于看视频的体验。因为他对文字足够熟悉，文字自带音频影像讯息。但初学者就不同了。初学者阅读时，要先花很大的力气，在脑海中把文字翻译成意念。就像阅读外文原著一样，是隔着一层的。
2. 视频可以模仿。而模仿是大脑最熟悉的思维方式。婴儿最初就是通过模仿来学习语言的。
3. 某些具体的细节，只能用视频才能精确传达，比如颜色和声音。所以在特定领域比如游戏视频攻略，厨艺教学，视频要比文字好用多了。当然反过来，一些抽象的概念只能通过文字来传递。所谓眼耳鼻舌身意，六种感官的表达，视频和文字各有所长。至少在
脑机
接口成熟之前，大家都有各自的使用场景，谁也淘汰不了谁。
4. 视频和音频的信息密度比较低。视频一分钟大概有200多个字，但书籍每分钟500多个。初学者更容易适应前者这种比较慢的节奏。
5. 再就是很多视频自带娱乐属性，不用一直盯着看，更加符合广大
社畜
的生活节奏。比如我经常一遍做饭，一边听B站的睡前消息。
6. 哦还有用户习惯的问题。现代人听到新名词后的第一反应不是翻书，而是搜索。而更年轻的群体，第一反应往往是搜视频。
…
当然，视频无法浏览，无法检索和快速定位，这确实是硬伤。书籍只要随手一翻，就知道大概的内容了，哪些地方可以浏览，那些地方需要重点阅读。视频行吗？不行，只能从头看起。看完才发现，10分钟的视频，只有1分钟是想听的。
读书读到精彩之处，可以停下来慢慢琢磨，反复读上几遍。视频行吗？行，但麻烦，如果要回到10秒前，多拖几次播放轴，保证让你心情烦躁。
总之，新手入门适合用视频来熟悉场景，明确问题的输入和输出。但如果是进阶学习，深挖抽象概念，看书的效率就高得多了。
