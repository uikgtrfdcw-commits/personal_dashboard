# 中国经济奇迹到底有多少含金量？

- 日期: 2025-07-07
- 赞同: 0
- 评论: 0

---

我们常常用“奇迹”来形容改革开放后的经济增长。然而，只有将中国与那些曾经改变世界格局的经济体放在一起比较，才能看到中国经济增长速度究竟处在何种位置，其“奇迹”的含金量到底如何。
比较的起点自然是18至19世纪，工业革命时期的英国。作为现代社会的开创者，英国在经济年均增长率通常在百分之二左右徘徊，人均增长常常不足百分之一。看似平平无奇，但在当时可是前无古人的成就——人类首次摆脱几千年来的停滞，逃离马尔萨斯陷阱。19世纪也因此被称为是日不落帝国的世纪。
随之而来的是19世纪后半叶的“追赶者”——德国和美国，他们拥有了英国所不具备的优势——一个可以模仿的榜样。它们不必从零开始发明铁路和工厂，而是可以直接引进、消化并加以改良。这使得它们的工业化进程显著加速。在德国，凭借对科学教育的重视和国家力量的推动，其经济年增长率达到了近百分之三，在化工、电力等新兴领域还超越了英国。而美国凭借广袤的国土、丰富的资源和源源不断的移民，其年增长率高达百分之四左右。它们惊人的发展速度，也初步揭示了“后发优势”的巨大潜力。
第二次世界大战之后，经济增长速度的记录再次被刷新。在欧洲，满目疮痍的土地上迎来了“战后黄金时代”。西德的“经济奇迹”和法国的“光辉三十年”见证了年均百分之五到百分之八的增长率。这股增长浪潮的背后是战后重建的巨大需求、美国技术的加速扩散以及消费社会的兴起。这意味着即使是成熟的工业经济体，在特定历史条件下也能爆发出惊人的增长潜力。
真正为中国登场铺好舞台的，是日本和韩国在20世纪下半叶书写的东亚神话。它们将“后发优势”的剧本演绎到了极致。通过将国家战略与全球市场紧密结合，它们创造了持续数十年的、年均增长接近百分之十的惊人纪录。这在当时被认为是经济发展速度的极限，是后发展国家所能达成的最高成就。
当中国在1978年打开大门时，它继承了所有先行者的经验与教训，站在了东亚奇迹的肩膀上。随后发生的故事我们就都知道了。中国几乎完美地复制了东亚经济体接近两位数的年均增长率，而中国的独特之处是无可比拟的经济体量。日本和韩国是在数千万人口的规模上创造了经济增长，而中国，是在超过十亿人口的土地上掀起了这场规模空前的工业化浪潮。和韩国一样，这场近百分之十的超高速增长周期持续了三十年之久。
因此，当我们回到最初的问题，答案也随之清晰。将中国的增长置于历史长河中，我们看到的是人类文明薪火相传的接力赛。从英国缓慢而坚定的起步，到德美两国的加速追赶，再到战后欧洲的强劲复苏和日韩的全力冲刺，世界经济增长的“速度极限”在不断被刷新。而中国的非凡之处在于，它不仅达到了这个极限速度，并且是以一种前所未有的巨大规模完成的这一切。 
人均GDP
然而在这前无古人的成就之下，是一个不容忽视的现实：尽管取得了持续时间最长、规模最大的高速增长，但时至今日，中国的人均财富水平仍然远远低于日本、韩国等“前辈”。这个看似矛盾的现象，其实也证实了中国发展之路的无比艰辛。
首先，中国经济的“起跑线”极度落后。经历了几十年极左思潮的反复折腾，1978年的国民经济濒临崩溃，人均GDP为156美元，不及非洲平均水平（552美元）的三分之一，更不及当时世界平均水平（约1900美元）的十分之一，与美国（超10000美元）相比更是天壤之别。这个极低的起点，意味着中国需要付出加倍的努力、攀爬更长的距离，才能达到别人当年的起跑线。
此外，中国作为超大规模国家，在发展过程中面临的问题也远比小型经济体复杂。巨大的地域差异、悬殊的城乡差距、资源与环境的承载压力，以及确保一个统一大市场平稳运行并维护社会稳定的巨大成本，都是在讨论中国发展时必须考虑的制约因素。
最后，日韩早已完成了从经济追赶者到发达经济体的转型，作为“高收入俱乐部”的成员积累了数十年的财富。而中国则是这个俱乐部的“准会员”，六亿人月收入不足1000，而超高速增长的阶段已经结束，正面临着“中等收入陷阱”严峻考验。我们看到的，是一个资深雇员和一个“应届生”之间的财富存量差异。
因此，当前人均GDP的差距，非但没有削弱中国增长故事的传奇色彩，反而从另一个维度印证了其成就的来之不易。中国过去四十年的增长奇迹，是在克服了难以置信的不利起点、承受了巨大的人口与环境压力、并在他人已经遥遥领先的背景下取得的。中国经济的人均数字或许不够亮眼，但对于一个如此体量的国家，能从如此深重的困境中走到今天，这本身就已是世界经济史上了不起的篇章。
下半场
那么，当一个国家完成了追赶，抵达技术前沿的边缘时，接下来又该怎么办？
这是中国和其他所有追赶者共同面对的问题。后发国家的经济增长，本质上是沿着一张现成的地图按图索骥： 通过学习和模仿，释放巨大潜力，实现"追赶式增长"。然而，当一国接近世界技术前沿时，地图就到了尽头。简单的模仿已无以为继，增长的引擎必须切换为"自主创新"。这就像一场接力赛，中国已经漂亮地跑完了第一棒，但第二棒的赛道、规则和跑法都截然不同。
后一种增长模式，被经济学家熊彼特称为“创造性破坏”：新思想、新技术和新商业模式不断涌现，初创公司不断挑战行业巨头，而低效的旧企业则被淘汰出局。这个过程或许是痛苦的，但它却是经济保持活力、实现持续繁荣的根本动力。它要求整个社会从上到下，都形成一种拥抱不确定性、鼓励试错、容忍失败的制度和文化。
也就是说，要点燃这个强大的新引擎，需要一个与过去截然不同的制度环境。
首先，竞争必须是真正公平和开放的。这意味着市场的准入门槛要足够低，让任何一个有颠覆性想法的“无名之辈”，都有机会挑战那些实力雄厚、甚至与公权力关系密切的行业巨头。如果新来者担心自己的成果会被轻易窃取，或者被选择性的监管扼杀在摇篮，那么创新的火花就很难被点燃。
其次，需要产生真正技术突破的顶尖研究型大学。这不仅意味着天量的资金投入，更关键的是要给予大学和科研人员充分的自主权，让他们能够出于好奇心去探索未知，而不是仅仅为了完成指令性的指标考核。真正的颠覆性创新，往往诞生于那些最初看起来“无用”的自由探索之中。
此外，金融体系也需要与时俱进。传统的银行借贷偏爱国有资产和稳定的现金流，这对于高风险、高失败率的创新活动来说，几乎是不可逾越的障碍。创新需要的是一个能够拥抱风险、识别潜力、不干涉经营的股权融资体系，也就是发达的风险投资机构和信用评价体系。它们扮演的不是债主，而是与创业者共担风险的合伙人。
最后，当“创造性破坏”不可避免地带来旧岗位的消亡时，一个灵活而有保障的劳动力市场也必不可少。这套体系需要巧妙地结合两方面：一方面，企业在调整人员时要有足够的灵活性；另一方面，劳动者在面临失业时，要获得足够的社会保障、有效的再就业培训以非歧视性的就业环境。这种制度安排能大大降低社会对变革的恐惧和阻力，让整个经济体的新陈代谢更加顺畅。
这一切最终都指向一个更深层次的问题：政府的角色。在追赶阶段，政府可以扮演一个强有力的家长，集中力量办大事，修建基础设施，引进关键技术。但在创新阶段，它的角色需要转变为市场规则制定者、维护者和公共服务提供者。其核心职责不再是直接下场比赛或指定冠军，而是制定和维护一套公平、透明且稳定的游戏规则，保障每个参与者的平等权利。
只有当人们内心深处相信，规则是持久的、产权是受保护的、契约是会被尊重的，他们才敢于进行那些长期投入的、高风险的创新活动。这种稳定的预期，本质上是一种公共信任，是创新社会最宝贵的财富。
中国正处在一个关键的十字路口。能否成功地从一个擅长模仿和追赶的经济体，转型为一个鼓励试错和颠覆性创新的社会，将决定它能否真正跨越“中等收入陷阱”。这不仅是一场深刻的经济变革，更是一场关于社会活力与制度建设的巨大考验。
在这场漫长的经济发展接力赛中，中国已经出色地完成了上半场。至于下半场的表现究竟如何，全世界都将拭目以待。
