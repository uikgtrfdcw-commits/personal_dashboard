# 怎样的文笔才算好？

- 日期: 2020-06-30
- 赞同: 0
- 评论: 0
- 链接: https://www.zhihu.com/question/26395133/answer/1311529582

---

心法决定招式。一个人的文笔、风格、习惯是由写作动机决定的。而动机大体上可以分为两种： 
认识自我。对应的风格是对心思意念的真实表达，比如Natalie Goldberg。
探索这个世界。对应的的风格是对逻辑观点的精准捕捉，比如Paul Granham。
这两种动机是截然不同的。前者主张忠实记录自己的内心，让声音自由的翱翔。后者主张要尽量简短，不值得听的就不说。前者认为应当还原内心的初始意念，真实的东西最有力量。后者喜欢反复修改自己的文章。两种文风都有各自的倾向、意图和阅读乐趣，不好区分是非高下。
当然乱七八糟的动机也有很多。比如我最初写作的动机，是让自己简单的头脑跟得上同事的节奏... 因此我的风格是尽量简单。无论是捕捉最纯净的思考，还是表达最简明的感受，我都希望把理解成本降到最低。我喜欢清晰明确，讨厌云山雾罩，喜欢开门见山，讨厌无意义的口水话。
