# 为何人随着年龄的增大觉得时间过得越来越快？

- 日期: 2022-05-27
- 赞同: 0
- 评论: 0
- 链接: https://www.zhihu.com/question/30824529/answer/2503291449

---

送给你一颗子弹
小时候，一切都是新的，每一分钟都印象深刻，日子过的很慢
长大以后，每天只是重复的体验，光阴似箭，时光匆匆流转
小时候总是无能为力，就像大人的宠物一般
长大以后能改变更多事情，但内心已经不再勇敢
小时候，每个人都有一个长长的清单
它们被推迟，被搁置，被冷落在时间的阁楼上腐烂
长大以后，那些沉重、无奈、不得已的东西，却被当做生活本身来宣传
为什么明明无力前行，却还在肩负更多重担
为什么明明没有勇气，却总误以为没有时间
美好是时光飞逝，事后回忆，满是缤纷绚烂
空虚是度日如年，回首望去，十年如同一天
