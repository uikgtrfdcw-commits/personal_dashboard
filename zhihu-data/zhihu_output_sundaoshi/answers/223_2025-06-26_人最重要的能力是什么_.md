# 人最重要的能力是什么？

- 赞同: 0
- 评论: 0

预测。
我们都有过这种感觉，生活中的某些部分似乎相当稳定和可预测，而另一些部分则感觉像个赌场。比如，你经营一家咖啡馆，大致能猜到明天会有多少顾客。但如果你是个热衷短线交易的投机者，那么下一次操作有可能赚得盆钵满盈，也有可能亏得血本无归。
为什么会有这种区别？问题的关键在于：系统内部的元素是否会彼此影响。
当系统中的事件彼此独立时，系统是稳定的。咖啡馆的客流量就是这样——每个顾客独立决定今天要不要来喝咖啡，他们之间互不影响。当许多独立的小事件累加在一起时，极端的好运气和坏运气会相互抵消，最终结果总是被拉回到平均值附近。统计学上，这叫正态分布。在这个世界里，你可以安心地做计划，因为明天和今天不会差太多。（对数正态分布情况类似，相对而言变化更大，但同样是可预测的）
然而，当系统中的事件开始相互影响、彼此纠缠时，一切都变了。这种相互影响一旦形成循环，就会产生一种剧烈的放大效应——要理解它，最直观的例子是话筒啸叫。
在一个简单的、线性的、元素相互独立的的系统里，你对着话筒小声说，音箱就小声播放；你大声喊，音箱就大声播放。投入与产出成正比，一切都温和而可预测。
但当你把话筒对准音箱，魔法就发生了。一个微弱的声音从音箱发出，被话筒捕捉、放大，再更响亮地从音箱播放出来，然后又被话筒捕捉、再次放大……这个瞬间形成的循环，让一个微不足道的初始噪音，很快变成了一声震耳欲聋的尖啸。
在系统论中，这种现象叫做“正反馈循环”：系统的输出变成了输入的一部分，不断自我强化。在这里，一分耕耘不再等于一分收获，而可能是一千分、一万分的收获。这种现象有很多名字：链式反应、非线性增长、蝴蝶效应、左脚踩右脚、滚雪球。这最终导致资源高度集中，赢家通吃，输家颗粒无收。（幂律分布）
正反馈循环之所以如此极端，是因为它打破了前两种模式最重要的前提：独立性。在这个系统里，事件之间不再孤立，而是像多米诺骨牌一样相互影响、迅速扩散。
一旦你理解了这个机制，就会发现它在现实中无处不在。
想象一下股市泡沫是怎么形成的：你的同事告诉你他买了某只股票赚了钱，你开始心动，于是也买入；聚会时你把这个消息告诉朋友，他们也跟着买入；越来越多人买入推高了股价，而上涨的股价本身又成了"这只股票值得买"的证据，吸引更多人入场。价格就这样一圈一圈地自我强化，直到某一天，有人开始获利了结，同样的循环反向启动，踩踏式的崩盘随之而来。
社交媒体把这种效应放大到了极致。在舆论场上，一个话题冲上热搜的过程就是典型的正反馈循环。某件事刚开始被几个人讨论，你刷到之后觉得有趣，转发到群里；群友看到后也觉得有意思，又转发到自己的朋友圈；朋友圈里的人继续扩散……在传统媒体时代，一个热点的形成可能需要几天甚至几周；而在社交网络上，短短几个小时，一个原本无人问津的小事就能演变成全网热议的焦点——当然，它被遗忘的速度也同样快得惊人。
在这种由非线性规则主导的世界里，“蝴蝶效应”是常态。一个微小的、几乎无法察觉的初始优势——可能只是一次偶然的推荐，一个幸运的时间点，都可能在正反馈循环的放大下，瞬间变成决定性的胜负手。你无法预测下一首“神曲”会是什么，因为引爆流行的，往往只是某个主流媒体的随机转发。你也无法预测下一个塌方的明星是谁，因为一句无心之失，就可能导致炙手可热的大V一夜变凉。
这为我们提供了一种新的世界观。它告诉我们，许多戏剧性的结果，最初往往只是蝴蝶煽动了一下翅膀，经过系统内部那套看不见的、相互关联的、不断自我强化的作用机制导致的。认识到这一点，或许可以帮我们减少一些对精确预测的执念，转而开始去欣赏和敬畏那些意外的复杂之美。
这也为我们观察当下的世界提供了一个视角。我们今天在社交媒体上看到的情绪化、极端化的舆论场，以及某些新兴股票市场的非理性波动，相当程度上可以用正反馈循环来解释。在这些系统中并没有真正的独立事件，每个个体的思考和决策，都无时无刻不在受到周围舆论场的影响，很容易导致链式反应和极端化倾向。
这其实与美国"镀金时代"以散户为主导的早期股市非常相似，谣言和狂热是价格的唯一驱动力，暴涨暴跌成为常态。而今天的美股之所以相对成熟和理性，正是因为它经历了一个多世纪漫长而痛苦的"专业化"和"理性化"的过程。一代代投资者在一次次泡沫与崩溃中学会了敬畏风险，学会了独立分析，逐渐削弱了市场中"别人买我也买、别人卖我也卖"的羊群效应。
美股和投资者的共同成长，验证了这样一个规律：系统的长远发展，最终取决于其内部参与者独立决策能力的整体提升。因此，我们有理由相信，随着新一代网民和投资者在信息洪流中逐渐成熟，社交媒体和新兴市场终将褪去狂热，变得更加稳定和可预测。
注：
中文语境中，正反馈循环（positive feedback loop）容易被误解为"良性循环"（virtuous cycle），但前者本身是中性的——既能放大好事，也能放大坏事。与之相对的概念是负反馈循环（negative feedback loop），指系统自我纠偏、趋向稳定的机制。
此外，一些心理学译者会把正向激励（positive reinforcement）与正反馈循环（Positive feedback loop）混同。但前者是指心理学上的自我奖励，后者是系统论中的自我放大现象。
