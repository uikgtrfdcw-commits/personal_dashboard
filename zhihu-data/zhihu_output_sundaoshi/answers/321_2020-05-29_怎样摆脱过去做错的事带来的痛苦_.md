# 怎样摆脱过去做错的事带来的痛苦？

- 日期: 2020-05-29
- 赞同: 0
- 评论: 0
- 链接: https://www.zhihu.com/question/31783536/answer/1252073174

---

因为自己身上曾经发生过摧折人的事，所以特别能理解题主的感觉。 
如果痛苦是别人造成的那还好，但如果是因为自己做错了事，那么由此生出的内疚和自责，完全可以把人逼疯。罗素描述自己早年的抑郁状态时说，『那些因为自我厌恶而产生的痛苦，往往是灭顶之灾。』 
可谁又不会做错事呢？年轻人初入社会，碰的钉子不重，受的委屈不多，不知道自己的愚蠢，不了解实际的困难，遇见事横冲直撞，很容易懊悔终生。 
然而懊悔也好，失落也罢，生活总还是要继续的。一些人把痛苦埋藏在心里，故意不去想这件事，好像什么也未曾发生过。我当初也是一样。然而越是压抑自己的内心，当午夜梦回、万籁俱寂，不得不和自己的内心独处之时，我们就越是辗转难眠——如果当时换一种做法，那就就好了。 
所幸的是，时间到底能够疗愈一个人。日子久了，我们总会与自己和解。日子再久一些，我们甚至会领悟到痛苦给我们带来的某些好处。 
痛苦是力量的源泉。
当你已经身在谷底，就不会再轻易被阴暗的环境吓倒。当你已经历劫重生，就不会再在意生活中的小小磨难。当你已经一败涂地，就不会再畏首畏尾，因为你没有什么可输的。
用乔布斯的话说：『曾被苹果公司炒鱿鱼是我这辈子发生的最棒的事情。』
痛苦让我们意识到，什么是对自己真正重要的东西。因为只有当失去的时候，我们才会意识到自己曾经的拥有并非理所当然。痛苦会拨开我们心中嫉妒和贪婪的重重迷雾，指明前进的方向。痛苦让我们重新认识自己。 
痛苦让我们意识到自己不可能得到一切，所以会更加了解自己到底要过怎样的生活，并愿意付出相应的代价。 
我非常喜欢上面Jennyyy的回答。 
非凡的痛苦意味着非凡的能力。心里有想不开的事儿，意味着即将有某种突破。生活中遇到的大麻烦，长远来看都意味着某种使命。 
比如说，鼓励那些同样被生活压垮的年轻人，帮助他们重新找到追求自我的信念和勇气。
要相信自己所热爱的事物，坚持做下去，它便带你到你需要去的地方。
每当我回想起那些因为逃避选择，而在游戏中寻求慰籍的日子，那些在绝望和麻木中挣扎的日夜，我都会感到自己作为过来人，对那些和我当年处境相似的年轻一代，负有某种义不容辞的责任。帮助他们，就是帮助曾经的自己。 
所以一切都是有意义的。
拥抱自己的过去，从黑暗中汲取力量。感谢胸中的伤口，那是光照进来的地方。
