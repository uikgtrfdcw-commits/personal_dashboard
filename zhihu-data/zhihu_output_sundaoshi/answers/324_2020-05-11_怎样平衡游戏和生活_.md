# 怎样平衡游戏和生活？

- 日期: 2020-05-11
- 赞同: 0
- 评论: 0
- 链接: https://www.zhihu.com/question/267213654/answer/1216549412

---

作为一个在游戏生活平衡的问题上挣扎多年的老玩家，我对这个问题相当有心得… 
总结起来有几点：
1. 要有一个比打游戏更热爱，更能获得认同感的事情。
它会成为你生活的支点，为你提供成就感，帮你抵抗生活中的负面情绪。比如我喜欢读书写作，一想到可能有人点赞评论，我就会很有动力关掉电脑，去咖啡馆看书。
当然，热爱自己的本职工作是最理想的，但这可遇而不可求。
人要发展一些『非职业的兴趣』，也就是业余活动。往往他的业余活动比他的职业还更重要，因为一个人成就怎样，往往要看他闲暇时间利用的如何。
尤其是在组织尚不健全的中国社会，职业不容易适合我们的性情，我们要想生活不苦痛不堕落，只有多方发展。而有了业余时间心爱的玩艺，你就算做八个钟头抹桌子工作也不会感觉烦闷了。
中国人普遍有两种心态。一种追求上进，性格要强，不重视、甚至压抑自己真实的兴趣和感受；另一种是上进失败，头破血流，麻木了，只要活着就是好的，活成什么样子无所谓。从一些电影的名字就可以看出来：活着、找乐。这两种心态其实是一样的，都认为自己的内心世界毫无价值。
这大概也是为什么中国人过去那么热爱麻将。因为自己的时间没事干，就要打发时间，把过剩的精力消磨掉。兴趣爱好是什么玩意儿，赚钱吗？
思想文化的阵地我们不占领敌人就会占领。不能从创造性的工作中自我实现，就一定会在享乐中迷失自我。朱光潜说，如果没有一种正当的嗜好，没有一种在闲暇时可以寄托心神的东西，将来就要被恶习引诱了。
2. 要保持一个快乐的心情。
快乐的人才能唱出好听的歌，快乐的人才能做出好吃的菜，快乐的人能才能打好游戏。而心情不好的时候，就会容易掉分... 
而且心情不好的时候，更容易放纵自己。内耗严重，就没有足够的算力来理性思考。
快乐就是思维充分活跃，充分感受到自己生命力的状态。这通常和那些生命中有趣的发现有关。当你理解了某样困扰了你很久的东西，感到恍然大悟，于是感到平静、投入和快乐。
快乐和你打不打游戏没有必然关系。打游戏的时候的确比较容易全心投入，但也不一定总是这样。有时候你觉得这游戏太有意思了我能玩一辈子，有的时候你只是习惯性的打开游戏，感觉麻木、枯燥，但又停不下来。
“打游戏才能快乐”是一种执念，这很大程度上是小时候家里给憋的。因为精神上乏善可陈，营养缺乏，所以随便遇到一点有意思的东西，疯掉了，直接狼吞虎咽下去。久而久之，遇到真正的好东西也不会享受了。
丰富自己的生活，培养更多的爱好，发现一个更大的世界。实在没有时间，玩一玩不熟悉的游戏也是好的。毕竟，你要见多识广，才知道自己真正喜欢的是啥。
4. 寻找自己内心的声音，而不是被外界的感官刺激淹没。
主动思考和输出，否则你只是在打发时间而已。
不要按照他人的指示生活。很多人喜欢打游戏，是因为只有在游戏中他们才有自己的想法和选择。
5. 要有耐心。
并没有一个『对的方法』，能够让你一劳永逸的解决问题。很多时候问题只是冰山一角，下面是整个底层认知结构的失衡。你必须不断感知、学习和运营，做好持久战的准备。
罗素说，一个能够享受生活中的各种美好事物而不带来消极后果的人，就是一个道德高尚的人。你可以玩游戏，而且可以玩的很好。自由和美是人类永恒的追求。但是，任何追求都要付出努力的代价，游戏也不例外。
...
最后，关于戒游戏的问题...
反正我个人已经理解并接纳自己是个游戏宅的事实，不再纠结试图改变自己了。
慢慢磨合出了一整套工作生活打游戏的日程习惯，学会了做减法，对自己的要求变低了...
以前工作效率低，压力大，打游戏更多是发泄情绪，后来渐渐摸出门道，压力变小了，打游戏更多是出于喜爱。
其实和戒撸差不多。刚开始觉得不行太low，后面戒不掉就开始佛系，后面习惯了好像还挺enjoy的...
再说游戏也是缓解压力的方式好吧。和
止痛剂
一个道理。你光把止痛药停了，压力的源头还在怎么办，学上一代吃喝嫖赌吗？
我觉得打游戏还健康一点...
怎么说呢，生活是一个多变量、非线性的复杂系统。没有简单答案，只能慢慢调整摸索。总想一刀切，一步到位，往往会绕更大的弯路。
不要太绝对，世上哪有什么非黑即白的事。存在的事情自然有存在的道理，找到与之共存的方法就好啦。
对自己耐心一些。慢慢就会找到平衡点的。
